基礎中的基礎，但在不同參數組合下能發揮強大功能。

### `ls` - 列出目錄內容

|參數|說明|
|---|---|
|`-l`|以詳細清單模式顯示 (權限、擁有者、大小、時間)|
|`-a`|顯示所有檔案，包含隱藏檔 (以 `.` 開頭的檔案)|
|`-h`|以人類可讀格式顯示檔案大小 (KB, MB, GB)|
|`-t`|依修改時間排序 (最新的在最上面)|

**常用組合：**

```bash
ls -lah   # 顯示所有檔案詳細資訊，並以易讀格式顯示大小
```

### `cd` - 切換目錄

```bash
cd ~      # 回到使用者家目錄 (/home/username)
cd -      # 回到「上一個」所在的目錄 (非常實用)
cd ..     # 回到上一層目錄
```

### `mkdir` - 建立目錄 (Make Directory)

|參數|說明|
|---|---|
|`-p`|**建立多層目錄** (Parents)，若父目錄不存在會自動建立，不會報錯|
|`-m`|設定權限 (Mode)，例如 `-m 755`|

**常用情境：**

```bash
mkdir new_folder          # 建立一個新資料夾
mkdir -p project/src/css  # 一次建立多層結構 (常用！)
```

### `cp` - 複製檔案或目錄

|參數|說明|
|---|---|
|`-r`|**遞迴複製**，複製目錄時**必須**加此參數|
|`-a`|**歸檔模式**，保留連結、權限、時間戳記等所有屬性 (備份時常用)|
|`-v`|顯示詳細的複製過程|
|`-i`|覆蓋前先詢問 (防止誤刪)|

**常用情境：**

```bash
cp file.txt backup.txt          # 複製檔案
cp -r /source/folder /dest/     # 複製整個資料夾
cp -av /var/www/html /backup/   # 完整備份網站目錄，保留權限與時間
```

### `mv` - 移動或重新命名

```bash
mv old_name.txt new_name.txt    # 重新命名
mv file.txt /tmp/               # 移動檔案
```

### `rm` - 刪除檔案或目錄 (⚠️ 危險指令)

|參數|說明|
|---|---|
|`-r`|遞迴刪除 (用於刪除目錄)|
|`-f`|**強制刪除**，不詢問確認 (Force)|
|`-i`|刪除前逐一詢問|

**常用情境：**

```bash
rm file.txt           # 刪除檔案
rm -rf /path/folder   # 強制刪除整個資料夾 (請務必確認路徑正確！)
```
