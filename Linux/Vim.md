Vim 有四種主要模式：**一般模式 (Normal)**、**編輯模式 (Insert)**、**指令模式 (Command)** 與 **可視/選取模式 (Visual)**。

### 進入與退出

- `vim filename`：開啟檔案。
    
- **退出 (在一般模式下輸入)：**
    
    - `:q`：離開 (若無修改)。
        
    - `:q!`：強制離開不存檔。
        
    - `:wq` (或單純按 `ZZ`)：存檔並離開。
        

### 模式切換

- `Esc`：**回到一般模式 (最重要，迷路就按 Esc)**。
    
- **進入編輯模式 (Insert)：**
    
    - `i`：從游標處開始輸入 (Insert)。
        
    - `a`：從游標後一格開始輸入 (Append)。
        
    - `o`：在下方新增一空白行並進入編輯。
        
    - `O` (大寫)：在上方新增一空白行並進入編輯。
        
- **進入可視模式 (Visual / 選取字元)：**
    
    - `v`：一般字元選取 (搭配方向鍵選取文字)。
        
    - `V` (大寫)：整行選取。
        
    - `Ctrl + v`：區塊選取 (超好用！可用來一次註解多行)。
        

### 快速游標移動 (一般模式)

忘記方向鍵吧，熟練這些按鍵能讓你鍵步如飛。

- **基礎移動：** `h` (左)、`j` (下)、`k` (上)、`l` (右)。
    
- **單字跳躍：**
    
    - `w`：跳到下一個單字的字首。
        
    - `b`：跳回上一個單字的字首 (Back)。
        
    - `e`：跳到單字的字尾 (End)。
        
- **行內移動：**
    
    - `0` (數字零)：跳到該行的最前面。
        
    - `^`：跳到該行第一個「非空白」字元。
        
    - `$`：跳到該行的最後面。
        
- **段落與翻頁：**
    
    - `{` 和 `}`：跳到上一段 / 下一段 (以空白行分隔)。
        
    - `Ctrl + f`：往下翻一頁 (Page Down / Forward)。
        
    - `Ctrl + b`：往上翻一頁 (Page Up / Backward)。
        
- **畫面快速定位：**
    
    - `H`：游標跳到畫面頂端 (High)。
        
    - `M`：游標跳到畫面中間 (Middle)。
        
    - `L`：游標跳到畫面底部 (Low)。
        
- **檔案頭尾與搜尋：**
    
    - `gg`：跳到檔案第一行。
        
    - `G` (大寫)：跳到檔案最後一行。
        
    - `15G`：直接跳到第 15 行。
        
    - `/keyword`：向下搜尋關鍵字 (按 `n` 找下一個，`N` 找上一個)。
        

### 選擇、複製、剪下與貼上 (一般模式)

在 Vim 中，複製稱為 **Yank (y)**，刪除/剪下稱為 **Delete (d)**，貼上稱為 **Paste (p)**。

**方法一：結合可視模式 (視覺化操作，推薦新手)**

1. 按 `v` 或 `V` 進入選取模式，用方向鍵反白你想操作的文字。
    
2. 按 `y` 複製，或按 `d` 剪下。
    
3. 移動游標到目的地，按 `p` 貼上。
    

**方法二：快捷指令 (熟練後速度極快)**

- **複製 (Yank)：**
    
    - `yy`：複製游標所在的那一整行。
        
    - `3yy`：往下複製 3 行 (包含目前這行)。
        
    - `yw`：複製一個單字。
        
    - `y$`：複製從游標處到該行結尾的所有文字。
        
- **剪下/刪除 (Delete)：**
    
    - `dd`：剪下/刪除一整行。
        
    - `5dd`：往下剪下/刪除 5 行。
        
    - `dw`：刪除一個單字。
        
    - `D` (或 `d$`)：刪除游標處到該行結尾的所有文字。
        
    - `x`：刪除游標所在的單一字元 (相當於 Delete 鍵)。
        
- **貼上 (Paste)：**
    
    - `p` (小寫)：貼在游標之**後** (如果是整行，則貼在下一行)。
        
    - `P` (大寫)：貼在游標之**前** (如果是整行，則貼在上一行)。
        
- **復原與重做：**
    
    - `u`：復原上一步 (Undo)。
        
    - `Ctrl + r`：重做 (Redo)。
        

### 編碼處理 (Encoding) - 解決亂碼問題

當打開 Windows 傳來的 Big5 檔案發現亂碼時，或需要轉檔時使用。

- **查看目前編碼：** `:set fileencoding`
    
- **以指定編碼「重新讀取」 (解決亂碼)：** `:e ++enc=big5` (或是 cp950) _這指令告訴 Vim：「請用 Big5 來解釋這個檔案」，亂碼通常會消失。_
    
- **轉換檔案編碼 (轉存)：** `:set fileencoding=utf-8` _設定好後，執行 `:w` 存檔，檔案就會變成 UTF-8 格式。_
