## 1. 核心概念：孤立 vs. 耦合（Isolated vs. Coupled）

| 類型 | 形式 | 意義 |
|---|---|---|
| 單一 ODE | $y' = f(t, y)$ | 描述**一個**孤立變數的動態 |
| ODE 系統 | $\mathbf{y}' = \mathbf{A}\mathbf{y} + \mathbf{g}$ | 描述**多個**變數的交互（電流耦合、族群互動、結構振動） |

> 在工程中，變數幾乎從不孤立存在：電流互相耦合、族群彼此影響、結構一起振動。ODE 系統正是描述這些「耦合」關係的數學語言。

**一般形式（General Form）：**

$$\mathbf{y}' = \mathbf{A}\mathbf{y} + \mathbf{g}$$

- **A**：常係數矩陣，代表系統的內部物理性質（system's internal physics）
- **g**：外部強迫向量（external forcing），如電壓源

## 2. 範例：電路網路（Example 2 — Electrical Network）

### 2.1 電路配置

| 元件 | 數值 |
|---|---|
| 電感 L | 1 henry |
| 電阻 R₁ | 4 ohms |
| 電容 C | 0.25 farad |
| 電阻 R₂ | 6 ohms |
| 電壓源 E | 12 volts |

初始條件：$I_1(0) = 0$，$I_2(0) = 0$（開關在 t = 0 閉合）

### 2.2 建立數學模型（Step 1）

對兩個迴路應用 **KVL（克希荷夫電壓定律）**：

**迴路 1：**
$$12 - I_1' - 4(I_1 - I_2) = 0$$

**迴路 2：**
$$4(I_1 - I_2) - 4\int I_2\, dt - 6I_2 = 0$$

整理後得：

$$I_1' = -4I_1 + 4I_2 + 12$$

$$I_2' = 0.4I_1' - 0.4I_2 = -1.6I_1 + 1.2I_2 + 4.8$$

寫成矩陣形式：

$$\mathbf{J}' = \mathbf{A}\mathbf{J} + \mathbf{g}, \quad \mathbf{J} = \begin{bmatrix} I_1 \\ I_2 \end{bmatrix},\; \mathbf{A} = \begin{bmatrix} -4.0 & 4.0 \\ -1.6 & 1.2 \end{bmatrix},\; \mathbf{g} = \begin{bmatrix} 12.0 \\ 4.8 \end{bmatrix}$$

> **注意**：矩陣 **A** 正是 4.0 節範例 1 的同一矩陣，特徵值為 λ₁ = −2、λ₂ = −0.8。

### 2.3 求解齊次系統（Step 2）

先解 $\mathbf{J}' = \mathbf{A}\mathbf{J}$，代入試解 $\mathbf{J} = \mathbf{x}e^{\lambda t}$，化為特徵值問題 $\mathbf{A}\mathbf{x} = \lambda\mathbf{x}$。

由 4.0 節結果：

$$\lambda_1 = -2,\; \mathbf{x}^{(1)} = \begin{bmatrix}2\\1\end{bmatrix}; \qquad \lambda_2 = -0.8,\; \mathbf{x}^{(2)} = \begin{bmatrix}1\\0.8\end{bmatrix}$$

齊次通解：

$$\mathbf{J}_h = c_1 \begin{bmatrix}2\\1\end{bmatrix}e^{-2t} + c_2\begin{bmatrix}1\\0.8\end{bmatrix}e^{-0.8t}$$

### 2.4 求特解（Particular Solution）

試 $\mathbf{J}_p = \mathbf{a}$（常數向量），代入得 $\mathbf{0} = \mathbf{A}\mathbf{a} + \mathbf{g}$，解方程組：

$$-4.0a_1 + 4.0a_2 + 12.0 = 0$$
$$-1.6a_1 + 1.2a_2 + 4.8 = 0$$

解得 $\mathbf{a} = \begin{bmatrix}3\\0\end{bmatrix}$

### 2.5 套用初始條件

全解 $\mathbf{J} = \mathbf{J}_h + \mathbf{J}_p$，代入 $I_1(0) = 0$，$I_2(0) = 0$：

$$c_1 = -4,\quad c_2 = 5$$

### 2.6 最終結果

$$\boxed{I_1(t) = -8e^{-2t} + 5e^{-0.8t} + 3}$$

$$\boxed{I_2(t) = -4e^{-2t} + 4e^{-0.8t}}$$

**物理詮釋：** 兩個特徵值均為負數（−2 和 −0.8），暫態電流指數衰減，最終 $I_1 \to 3\,\text{A}$，$I_2 \to 0\,\text{A}$。系統為**穩定（stable）**。

## 3. 高階 ODE 轉換定理（Theorem 1）

### 定理內容

一個 n 階 ODE：

$$y^{(n)} = F(t,\, y,\, y',\, \ldots,\, y^{(n-1)})$$

令 $y_1 = y,\; y_2 = y',\; y_3 = y'',\; \ldots,\; y_n = y^{(n-1)}$，可轉換為 **n 個一階 ODE 組成的系統**：

$$y_1' = y_2$$
$$y_2' = y_3$$
$$\vdots$$
$$y_{n-1}' = y_n$$
$$y_n' = F(t,\, y,\, y',\, \ldots,\, y^{(n-1)})$$

**意義：** 任何高階 ODE 都可以化為一階系統，統一用矩陣方法處理。

## 4. 範例：彈簧質量系統（Example 3 — Mass on a Spring）

### 4.1 原始方程式

$$my'' + cy' + ky = 0 \implies y'' = -\frac{c}{m}y' - \frac{k}{m}y$$

### 4.2 轉換為一階系統

令 $y_1 = y$，$y_2 = y'$：

$$y_1' = y_2$$
$$y_2' = -\frac{k}{m}y_1 - \frac{c}{m}y_2$$

矩陣形式：

$$\mathbf{y}' = \mathbf{A}\mathbf{y} = \begin{bmatrix} 0 & 1 \\ -k/m & -c/m \end{bmatrix}\begin{bmatrix}y_1\\y_2\end{bmatrix}$$

特徵方程式：

$$\lambda^2 + \frac{c}{m}\lambda + \frac{k}{m} = 0$$

> 這與第二章直接對 $my'' + cy' + ky = 0$ 求特徵方程式的結果完全一致！轉換前後的特徵值相同，驗證了方法的一致性。

### 4.3 具體數值（m = 1, c = 2, k = 0.75）

$$\lambda^2 + 2\lambda + 0.75 = (\lambda + 0.5)(\lambda + 1.5) = 0$$

$$\lambda_1 = -0.5,\; \mathbf{x}^{(1)} = \begin{bmatrix}2\\-1\end{bmatrix}; \qquad \lambda_2 = -1.5,\; \mathbf{x}^{(2)} = \begin{bmatrix}1\\-1.5\end{bmatrix}$$

通解（取 $y = y_1$）：

$$y(t) = 2c_1 e^{-0.5t} + c_2 e^{-1.5t}$$

## 5. 解題流程總結

```
工程問題（電路 / 機械 / 生態…）
        ↓ 套用物理定律（KVL、牛頓第二定律…）
  建立 ODE 系統：J' = AJ + g
        ↓
  解齊次系統 J' = AJ
  （求特徵值 λ、特徵向量 x → 通解 Jh）
        ↓
  求特解 Jp
  （待定係數法：試常數向量 / 指數函數…）
        ↓
  全解：J = Jh + Jp
        ↓
  代入初始條件，求常數 c₁, c₂, …
```

## 重點整理

| 概念 | 說明 | 關鍵公式 / 注意事項 |
|---|---|---|
| ODE 系統一般形式 | 描述多變數耦合動態 | $\mathbf{y}' = \mathbf{A}\mathbf{y} + \mathbf{g}$ |
| 齊次系統試解 | 代入 $\mathbf{y} = \mathbf{x}e^{\lambda t}$ 化為特徵值問題 | $\mathbf{A}\mathbf{x} = \lambda\mathbf{x}$ |
| 特解（常數強迫項） | 試常數向量 $\mathbf{J}_p = \mathbf{a}$ | 令 $\mathbf{J}_p' = \mathbf{0}$，解 $\mathbf{A}\mathbf{a} + \mathbf{g} = \mathbf{0}$ |
| 高階 ODE 轉換 | n 階 → n 個一階系統 | $y_1 = y,\, y_2 = y',\, \ldots,\, y_n = y^{(n-1)}$ |
| 穩定性快速判斷 | 所有特徵值實部 < 0 → 暫態項衰減 → 穩定 | 特徵值由矩陣 **A** 決定，與 **g** 無關 |
