## 1. 數學建模的四大步驟 (Modeling Cycle)

利用微分方程式解決實際問題時，通常會經過以下流程 ：

- **步驟一：** 觀察實際的物理系統 (Physical systems) 。
    
- **步驟二：** 建立對應的數學模型 (Mathematical model) 。
    
- **步驟三：** 運算並求出數學解 (Mathematical solution) 。
    
- **步驟四：** 對結果進行物理解釋 (Physical interpretation) 。
    

## 2. 常微分方程式 (ODE) 的基本定義

- **常微分方程式 (ODE)：** 方程式中包含一個未知函數（通常稱為 $y(x)$）以及它的一個或多個導數 。
    
- **階數 (Order)：** 方程式中未知函數 $y$ 的最高階導數 $n$，即為該微分方程式的階數 。
    
- **一階 ODE 的隱式形式 (Implicit form)：** $F(x, y, y') = 0$ 。
    
- **一階 ODE 的顯式形式 (Explicit form)：** $y' = f(x, y)$ 。
    
- **解的概念 (Concept of solution)：** 如果一個函數 $y = h(x)$ 在某個開區間 $a < x < b$ 內有定義且可微分，並且將 $h$ 和 $h'$ 代入方程式後能使其成為恆等式，則該函數即為此 ODE 的解 。
    

## 3. 微分方程式解的種類

- **通解 (General solution)：** 解的表達式中包含一個任意常數 $c$ 。
    
- **特解 (Particular solution)：** 解的表達式中不包含任何任意常數 。
    
- **奇解 (Singular solution)：** 有些 ODE 存在無法從通解中取得的額外解，稱為奇解 。例如方程式 $y'^2 - xy' + y = 0$ 除了通解 $y = cx - c^2$ 之外，還具有奇解 $y = x^2/4$ 。
    
- **初始值問題 (Initial Value Problem, IVP)：** 包含一個微分方程式以及一個初始條件，一般寫作 $y' = f(x, y)$ 且 $y(x_0) = y_0$ 。
    

## 4. 簡報中提到的經典物理模型範例

簡報列舉了許多未來會遇到的系統模型：

- **自由落體 (Falling stone)：** 模型為 $y'' = g = const$，代表等加速度運動 。
    
- **跳傘員 (Parachutist)：** 模型為 $mv' = mg - bv^2$，代表具有空氣阻力的加速運動 。
    
- **水庫水位 (Water level h)：** 模型為 $h' = -kh$，代表托里切利定律 (Torricelli’s law) 。
    
- **彈簧質量系統 (Vibrating mass on a spring)：** 模型為 $my'' + ky = 0$，代表簡諧振盪 。
    
- **RLC 電路 (RLC circuit)：** 模型為 $LI'' + RI' + \frac{1}{C}I = E'$ 。
    
- **指數衰減 (Exponential Decay) 與放射性：** 物理過程的數學模型可表示為 $\frac{dy}{dt} = -ky$，若初始值 $y(0) = 0.5$，則數學解為 $y(t) = 0.5e^{-kt}$ 。其物理解釋為該物質的量會隨時間遞減（因為 $k$ 為正值），且當時間趨近於無限大時，其極限值為零 。
