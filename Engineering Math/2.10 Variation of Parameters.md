# 2.10 Solution by Variation of Parameters

## 方法概覽

**Variation of Parameters**（參數變換法）是求 nonhomogeneous ODE particular solution
的通用方法，由 Lagrange 提出。相較於 undetermined coefficients，此方法適用範圍更廣，
即使 $r(x)$ 是 $\sec x$、$\ln x$ 等無法查表的函數也能處理。

---

## 核心公式

給定 nonhomogeneous linear ODE：

$$y'' + p(x)y' + q(x)y = r(x)$$

設 $y_1$、$y_2$ 為對應 homogeneous ODE 的一組 basis，Particular solution 為：

$$\boxed{y_p(x) = -y_1 \int \frac{y_2\, r}{W}\,dx + y_2 \int \frac{y_1\, r}{W}\,dx}$$

其中 **Wronskian**：

$$W = y_1 y_2' - y_2 y_1'$$

---

## 公式推導

### 設定 $y_p$ 的形式

令：

$$y_p(x) = u(x)\,y_1(x) + v(x)\,y_2(x)$$

其中 $u(x)$、$v(x)$ 為待求函數（不再是常數，故稱「參數變換」）。

### 微分並施加額外條件

$$y_p' = u'y_1 + uy_1' + v'y_2 + vy_2'$$

為簡化計算，施加條件：

$$u'y_1 + v'y_2 = 0 \quad \cdots (1)$$

則 $y_p' = uy_1' + vy_2'$，繼續微分：

$$y_p'' = u'y_1' + uy_1'' + v'y_2' + vy_2''$$

### 代入原 ODE

代入 $y'' + py' + qy = r$ 並整理，利用 $y_1$、$y_2$ 各自滿足 homogeneous ODE，
得：

$$u'y_1' + v'y_2' = r \quad \cdots (2)$$

### 解聯立方程組

由 (1) 與 (2) 組成的線性方程組，係數行列式恰為 Wronskian $W$：

$$u' = -\frac{y_2\, r}{W}, \quad v' = \frac{y_1\, r}{W}$$

積分得：

$$u = -\int \frac{y_2\, r}{W}\,dx, \quad v = \int \frac{y_1\, r}{W}\,dx$$

代回 $y_p = uy_1 + vy_2$，即得核心公式。

> **為何需要 $W \neq 0$**：若 $W = 0$，方程組無解，方法失效。
> 這也是 2.6 節強調 basis 必須 linearly independent 的原因之一。

---

## 求解步驟

```
Step 1：求解 homogeneous ODE
  找出 basis y₁, y₂ → 得到 y_h = c₁y₁ + c₂y₂
         │
         ▼
Step 2：計算 Wronskian
  W = y₁y₂' - y₂y₁'
         │
         ▼
Step 3：代入 Lagrange 公式積分
  yₚ = -y₁∫(y₂r/W)dx + y₂∫(y₁r/W)dx
         │
         ▼
Step 4：組合 general solution
  y = y_h + yₚ = c₁y₁ + c₂y₂ + yₚ
         │
         ▼
Step 5：（若有 initial conditions）
  代入求 c₁, c₂ → particular solution
```

## 範例：$y'' + y = \sec x$

$$y'' + y = \sec x = \frac{1}{\cos x}$$

> $\sec x$ 不在 undetermined coefficients 查表範圍內，必須用 variation of parameters。

### Step 1：Homogeneous ODE 的 Basis

$$y'' + y = 0 \implies \lambda^2 + 1 = 0 \implies \lambda = \pm i$$

$$y_1 = \cos x, \quad y_2 = \sin x$$
$$y_h = c_1\cos x + c_2\sin x$$

### Step 2：計算 Wronskian

$$W = \begin{vmatrix} \cos x & \sin x \\ -\sin x & \cos x \end{vmatrix}
= \cos^2 x + \sin^2 x = 1$$

### Step 3：代入公式積分

$$r(x) = \sec x = \frac{1}{\cos x}, \quad W = 1$$

計算 $u$ 和 $v$：

$$u = -\int \frac{y_2\, r}{W}\,dx = -\int \sin x \cdot \sec x\,dx = -\int \tan x\,dx = \ln|\cos x|$$

$$v = \int \frac{y_1\, r}{W}\,dx = \int \cos x \cdot \sec x\,dx = \int 1\,dx = x$$

### Step 4：組合 Particular Solution

$$y_p = u\,y_1 + v\,y_2 = \cos x \ln|\cos x| + x\sin x$$

### 最終 General Solution

$$\boxed{y = c_1\cos x + c_2\sin x + \cos x\ln|\cos x| + x\sin x}$$

## 方法比較：Undetermined Coefficients vs. Variation of Parameters

| 比較項目 | Undetermined Coefficients | Variation of Parameters |
|:---|:---:|:---:|
| **適用範圍** | $r(x)$ 為指數、多項式、三角函數 | 任意連續的 $r(x)$ |
| **計算難度** | 較簡單（代數運算） | 較複雜（需積分） |
| **ODE 係數** | 必須為常數 | 可以是變數 $p(x)$、$q(x)$ |
| **限制** | $r(x)$ 需可查表 | 積分不一定有 closed form |
| **典型 $r(x)$** | $e^x$, $\sin x$, $x^2$ | $\sec x$, $\ln x$, $1/x$ |

### 選擇策略

```
分析 r(x)
    │
    ├─ r(x) = 0
    │       └─→ Homogeneous method（特徵方程）
    │
    └─ r(x) ≠ 0
            │
            ├─ 指數 / 多項式 / 三角函數（可查表）
            │       └─→ Method 1: Undetermined Coefficients（快速）
            │
            └─ sec x, ln x, tan x 等（不可查表）
                    └─→ Method 2: Variation of Parameters（通用）
```

## 本節核心重點整理

| 概念 | 說明 |
|:---|:---|
| **方法本質** | 將 homogeneous 解的常數 $c_1, c_2$ 換成函數 $u(x), v(x)$ |
| **核心公式** | $y_p = -y_1\int\frac{y_2 r}{W}dx + y_2\int\frac{y_1 r}{W}dx$ |
| **Wronskian 的角色** | 出現在公式分母，必須 $W \neq 0$ 方法才有效 |
| **額外條件** | $u'y_1 + v'y_2 = 0$，是推導時人為施加以簡化計算 |
| **通用性** | 只要 $r(x)$ 連續且 $y_1, y_2$ 已知，必能求出 $y_p$ |
| **與 2.6 的連結** | Wronskian 同時是 linear independence 的判斷工具 |
