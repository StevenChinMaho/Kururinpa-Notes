## 1. 週期 $2L$ 的 Fourier Series（Transition from Period $2\pi$ to Any Period $2L$）

### 1.1 公式推導動機

11.1 節的公式假設週期為 $2\pi$。實際工程問題中週期往往是任意值 $2L$。做變數代換 $v = \pi x / L$，可將週期 $2L$ 的問題化為週期 $2\pi$，再代回原變數。

### 1.2 一般化的 Fourier Series（週期 $2L$）

$$\boxed{f(x) = a_0 + \sum_{n=1}^{\infty} \left( a_n \cos\frac{n\pi x}{L} + b_n \sin\frac{n\pi x}{L} \right)}$$

### 1.3 Fourier Coefficients（週期 $2L$）

$$\boxed{a_0 = \frac{1}{2L} \int_{-L}^{L} f(x)\, dx}$$

$$\boxed{a_n = \frac{1}{L} \int_{-L}^{L} f(x) \cos\frac{n\pi x}{L}\, dx \qquad n = 1, 2, \ldots}$$

$$\boxed{b_n = \frac{1}{L} \int_{-L}^{L} f(x) \sin\frac{n\pi x}{L}\, dx \qquad n = 1, 2, \ldots}$$

> 注意：令 $L = \pi$ 即還原為 11.1 節的週期 $2\pi$ 公式。

---

### 1.4 Example 1：週期 $2L = 2$ 的矩形波

函數定義（$L = 1$）：

$$f(x) = \begin{cases} 0 & -1 < x < 0 \\ k & 0 < x < 1 \end{cases}, \quad f(x+2) = f(x)$$

計算 $a_n$：

$$a_n = \frac{1}{2} \int_{-2}^{2} f(x)\cos\frac{n\pi x}{2}\, dx = \frac{1}{2}\int_{-1}^{1} k\cos\frac{n\pi x}{2}\, dx = \frac{2k}{n\pi}\sin\frac{n\pi}{2}$$

$$a_n = \frac{2k}{n\pi} \quad (n = 1, 5, 9, \ldots), \qquad a_n = -\frac{2k}{n\pi} \quad (n = 3, 7, 11, \ldots)$$

結果：

$$f(x) = \frac{k}{2} + \frac{2k}{\pi}\left(\cos\frac{\pi}{2}x - \frac{1}{3}\cos\frac{3\pi}{2}x + \frac{1}{5}\cos\frac{5\pi}{2}x - + \cdots\right)$$

---

### 1.5 Example 2：Change of Scale（縮放週期）

週期 $2L = 4$（$L = 2$），對矩形波做變數代換 $v = \pi x / 2$：

原 $g(v)$ 的 Fourier Series（週期 $2\pi$）：

$$g(v) = \frac{4k}{\pi}\left(\sin v + \frac{1}{3}\sin 3v + \frac{1}{5}\sin 5v + \cdots\right)$$

代回 $v = \pi x / 2$，得到週期 $4$ 的 Fourier Series：

$$f(x) = \frac{4k}{\pi}\left(\sin\frac{\pi}{2}x + \frac{1}{3}\sin\frac{3\pi}{2}x + \frac{1}{5}\sin\frac{5\pi}{2}x + \cdots\right)$$

---

### 1.6 Example 3：Half-Wave Rectifier（半波整流器）

函數定義（週期 $2L = 2\pi/\omega$）：

$$u(t) = \begin{cases} E\sin\omega t & 0 < t < \pi/\omega \\ 0 & -\pi/\omega < t < 0 \end{cases}$$

計算 $a_0$：

$$a_0 = \frac{\omega}{2\pi}\int_{0}^{\pi/\omega} E\sin\omega t\, dt = \frac{E}{\pi}$$

計算 $a_n$（$n \neq 1$，利用積化和差）：

$$a_n = \frac{\omega E}{2\pi}\int_{0}^{\pi/\omega}[\sin(1+n)\omega t + \sin(1-n)\omega t]\, dt$$

$$a_n = \frac{E}{2\pi}\left(\frac{-\cos(1+n)\pi + 1}{1+n} + \frac{-\cos(1-n)\pi + 1}{1-n}\right)$$

對 $n = 2, 4, 6, \ldots$（偶數）：

$$a_n = -\frac{2E}{(n-1)(n+1)\pi}$$

最終 Fourier Series：

$$\boxed{u(t) = \frac{E}{\pi} + \frac{E}{2}\sin\omega t - \frac{2E}{\pi}\left(\frac{1}{1\cdot 3}\cos 2\omega t + \frac{1}{3\cdot 5}\cos 4\omega t + \cdots\right)}$$

> 物理意義：$E/\pi$ 為直流分量，$\frac{E}{2}\sin\omega t$ 為基頻分量，其餘為偶次諧波（even harmonics）。

## 2. Even / Odd 函數的化簡（Simplifications）

### 2.1 定義

| 類型 | 定義 | 圖形特徵 |
|---|---|---|
| **Even function（偶函數）** | $f(-x) = f(x)$ | 對稱於 $y$ 軸 |
| **Odd function（奇函數）** | $f(-x) = -f(x)$ | 對稱於原點 |

### 2.2 積分性質

對稱區間 $[-L, L]$ 上的積分：

$$\int_{-L}^{L} (\text{奇函數})\, dx = 0$$

$$\int_{-L}^{L} (\text{偶函數})\, dx = 2\int_{0}^{L} (\text{偶函數})\, dx$$

### 2.3 乘積的奇偶性

| $f$ | $g$ | $f \cdot g$ |
|---|---|---|
| 偶 | 偶 | 偶 |
| 奇 | 奇 | 偶 |
| 偶 | 奇 | 奇 |

> 因此：偶函數 $\times$ sin（奇）$=$ 奇函數，積分為零 → $b_n = 0$。
> 奇函數 $\times$ cos（偶）$=$ 奇函數，積分為零 → $a_n = 0$。

### 2.4 偶函數的 Fourier Series（Fourier Cosine Series）

若 $f(x)$ 為週期 $2L$ 的**偶函數**，則 $b_n = 0$，只有 cos 項：

$$f(x) = a_0 + \sum_{n=1}^{\infty} a_n \cos\frac{n\pi x}{L}$$

$$a_0 = \frac{1}{L}\int_{0}^{L} f(x)\, dx, \qquad a_n = \frac{2}{L}\int_{0}^{L} f(x)\cos\frac{n\pi x}{L}\, dx$$

### 2.5 奇函數的 Fourier Series（Fourier Sine Series）

若 $f(x)$ 為週期 $2L$ 的**奇函數**，則 $a_n = 0$，只有 sin 項：

$$f(x) = \sum_{n=1}^{\infty} b_n \sin\frac{n\pi x}{L}$$

$$b_n = \frac{2}{L}\int_{0}^{L} f(x)\sin\frac{n\pi x}{L}\, dx$$

---

### 2.6 Example 5：Sawtooth Wave（鋸齒波）

函數 $f(x) = x$，週期 $2\pi$，定義在 $(-\pi, \pi)$。

$f(x) = x$ 為**奇函數**，故 $a_n = 0$，只需計算 $b_n$：

$$b_n = \frac{2}{\pi}\int_{0}^{\pi} x \sin nx\, dx$$

利用分部積分：

$$b_n = \frac{2}{\pi}\left[\frac{-x\cos nx}{n}\Bigg|_{0}^{\pi} + \frac{1}{n}\int_{0}^{\pi}\cos nx\, dx\right] = -\frac{2}{n}\cos n\pi$$

結果（$\cos n\pi = (-1)^n$）：

$$b_n = \frac{2(-1)^{n+1}}{n} \implies b_1 = 2,\; b_2 = -1,\; b_3 = \frac{2}{3},\; \ldots$$

Fourier Series：

$$\boxed{f(x) = x + \pi = \pi + 2\left(\sin x - \frac{1}{2}\sin 2x + \frac{1}{3}\sin 3x - + \cdots\right)}$$

> 說明：若定義 $f(x) = x + \pi$（使函數在 $[0, 2\pi)$ 上為正），Partial sums $S_1, S_2, S_3, S_{20}$ 逐漸收斂，但在 $x = \pm\pi$ 附近仍有 Gibbs phenomenon。

## 3. Half-Range Expansions（半範圍展開）

### 3.1 問題設定

工程問題中，函數 $f(x)$ 有時只定義在 $[0, L]$ 上。此時可人為地將 $f$ 延伸為週期 $2L$ 的函數，有兩種選擇：

| 延伸方式 | 定義 | 結果 |
|---|---|---|
| **Even extension（偶延伸）** | $f_1(x) = f(-x)$ for $-L < x < 0$ | → Fourier Cosine Series |
| **Odd extension（奇延伸）** | $f_2(x) = -f(-x)$ for $-L < x < 0$ | → Fourier Sine Series |

> 兩種展開在 $[0, L]$ 上**都正確代表原始函數**，但在 $[-L, 0]$ 的行為不同。兩者在工程應用中各有用途（如偏微分方程邊界條件的選擇）。

### 3.2 Half-Range Cosine Series（偶延伸）

$$f(x) = a_0 + \sum_{n=1}^{\infty} a_n \cos\frac{n\pi x}{L}$$

$$a_0 = \frac{1}{L}\int_{0}^{L} f(x)\, dx, \qquad a_n = \frac{2}{L}\int_{0}^{L} f(x)\cos\frac{n\pi x}{L}\, dx$$

### 3.3 Half-Range Sine Series（奇延伸）

$$f(x) = \sum_{n=1}^{\infty} b_n \sin\frac{n\pi x}{L}$$

$$b_n = \frac{2}{L}\int_{0}^{L} f(x)\sin\frac{n\pi x}{L}\, dx$$

---

### 3.4 Example 6：三角波（"Triangle"）的 Half-Range Expansions

函數定義（$0 \leq x \leq L$）：

$$f(x) = \begin{cases} \dfrac{2k}{L}x & 0 \leq x \leq L/2 \\[6pt] \dfrac{2k}{L}(L - x) & L/2 \leq x \leq L \end{cases}$$

#### (a) Even Periodic Extension → Fourier Cosine Series

計算 $a_0$：

$$a_0 = \frac{1}{L}\left[\frac{2k}{L}\int_{0}^{L/2} x\, dx + \frac{2k}{L}\int_{L/2}^{L}(L-x)\, dx\right] = \frac{k}{2}$$

計算 $a_n$（積分後化簡）：

$$a_n = \frac{4k}{n^2\pi^2}\left(2\cos\frac{n\pi}{2} - \cos n\pi - 1\right)$$

非零項為 $n = 2, 6, 10, \ldots$（即 $n = 4m - 2$）：

$$a_2 = -\frac{16k}{2^2\pi^2}, \quad a_6 = -\frac{16k}{6^2\pi^2}, \quad a_{10} = -\frac{16k}{10^2\pi^2}, \quad \ldots$$

結果：

$$\boxed{f(x) = \frac{k}{2} - \frac{16k}{\pi^2}\left(\frac{1}{2^2}\cos\frac{2\pi}{L}x + \frac{1}{6^2}\cos\frac{6\pi}{L}x + \cdots\right)}$$

#### (b) Odd Periodic Extension → Fourier Sine Series

計算 $b_n$：

$$b_n = \frac{8k}{n^2\pi^2}\sin\frac{n\pi}{2}$$

結果：

$$\boxed{f(x) = \frac{8k}{\pi^2}\left(\frac{1}{1^2}\sin\frac{\pi}{L}x - \frac{1}{3^2}\sin\frac{3\pi}{L}x + \frac{1}{5^2}\sin\frac{5\pi}{L}x - + \cdots\right)}$$

> 兩種展開的係數衰減速率皆為 $1/n^2$（因為原函數連續），收斂速度比矩形波（$1/n$）快。

## 重點整理

| 概念 | 公式 / 條件 | 注意事項 |
|---|---|---|
| 週期 $2L$ 的 Fourier Series | $a_0 = \frac{1}{2L}\int_{-L}^{L}f\,dx$，$a_n = \frac{1}{L}\int_{-L}^{L}f\cos\frac{n\pi x}{L}\,dx$，$b_n = \frac{1}{L}\int_{-L}^{L}f\sin\frac{n\pi x}{L}\,dx$ | $L=\pi$ 時還原為 11.1 公式 |
| Even function | $f(-x) = f(x)$，只有 cos 項（$b_n = 0$） | 積分範圍可縮短為 $[0, L]$，再乘以 2 |
| Odd function | $f(-x) = -f(x)$，只有 sin 項（$a_n = 0$） | 積分範圍可縮短為 $[0, L]$，再乘以 2 |
| Sawtooth wave | 奇函數 → 只有 sin 項，$b_n = 2(-1)^{n+1}/n$ | Gibbs phenomenon 在不連續點出現 |
| Half-Range Cosine | 對 $[0,L]$ 函數做偶延伸，使用 cos 系列 | 係數公式：$a_n = \frac{2}{L}\int_{0}^{L} f\cos\frac{n\pi x}{L}\,dx$ |
| Half-Range Sine | 對 $[0,L]$ 函數做奇延伸，使用 sin 系列 | 係數公式：$b_n = \frac{2}{L}\int_{0}^{L} f\sin\frac{n\pi x}{L}\,dx$ |
| 選擇哪種延伸？ | 依邊界條件決定（PDE 應用中很關鍵） | 兩種展開在 $[0,L]$ 上皆正確 |
