# 11.3 Forced Oscillations（強迫振盪）

> Advanced Engineering Mathematics, Abridged Version, 10th Edition Update — Erwin Kreyszig, Wiley 2018

---

## 本節核心概念

Fourier Series 的最重要應用之一：當驅動力（driving force）不是單純的 sin / cos，而是**任意週期函數**時，利用 Fourier Series 將其分解，再對每一個諧波（harmonic）分別求解 ODE，最後疊加得到完整解。

---

## 1. 物理模型

### 1.1 機械振盪系統

質量–彈簧–阻尼系統（Mass–Spring–Dashpot），受外部週期力 $r(t)$ 驅動：

$$\boxed{my'' + cy' + ky = r(t)}$$

| 符號 | 意義 |
|---|---|
| $m$ | 質量（mass） |
| $c$ | 阻尼係數（damping coefficient） |
| $k$ | 彈簧常數（spring constant） |
| $r(t)$ | 外部驅動力（periodic driving force） |

### 1.2 電路系統（類比）

RLC 串聯電路，外部電壓源 $E(t)$ 驅動：

$$\boxed{LI'' + RI' + \frac{1}{C}I = E'(t)}$$

| 符號 | 意義 |
|---|---|
| $L$ | 電感（inductance） |
| $R$ | 電阻（resistance） |
| $C$ | 電容（capacitance） |
| $E(t)$ | 外部電壓源 |

> **機械 ↔ 電路類比**：$m \leftrightarrow L$，$c \leftrightarrow R$，$k \leftrightarrow 1/C$，$r(t) \leftrightarrow E'(t)$

---

## 2. 解題策略：Fourier Series + 疊加原理

### 步驟概覽

```
① 將週期驅動力 r(t) 展開為 Fourier Series
        ↓
② 對每一個諧波項（cos nt 或 sin nt）分別求 ODE 的特解 yₙ
        ↓
③ 利用疊加原理，將所有特解加總得到完整特解 y
        ↓
④ 加上齊次解（暫態響應），但若有阻尼則暫態會衰減消失
```

### 為何可以疊加？

ODE 為**線性**（linear），滿足疊加原理：若 $y_n$ 是對應 $r_n(t)$ 的解，則 $\sum y_n$ 是對應 $\sum r_n(t)$ 的解。

---

## 3. Example 1：非正弦週期驅動力的強迫振盪

### 問題設定

$$y'' + 0.05y' + 25y = r(t)$$

驅動力 $r(t)$ 為週期 $2\pi$、幅值 $\pi/2$ 的**鋸齒波（sawtooth-like triangle wave）**。

### Step 1：展開 $r(t)$ 為 Fourier Series

$r(t)$ 為奇函數（odd function），故只有 sin 項：

$$r(t) = \frac{4}{\pi}\left(\cos t + \frac{1}{3^2}\cos 3t + \frac{1}{5^2}\cos 5t + \cdots\right)$$

> 注意：此處 $r(t)$ 展開後含 **cos** 項（三角波為偶函數情況），每項形如：
> $$\frac{4}{n^2\pi}\cos nt \qquad (n = 1, 3, 5, \ldots)$$

### Step 2：對每個諧波求特解

對每個 $n = 1, 3, 5, \ldots$，求解：

$$y'' + 0.05y' + 25y = \frac{4}{n^2\pi}\cos nt$$

設特解形式為：

$$y_n = A_n \cos nt + B_n \sin nt$$

代入 ODE，利用待定係數法（method of undetermined coefficients）比較係數，得：

$$A_n = \frac{4(25 - n^2)}{n^2\pi D_n}, \qquad B_n = \frac{0.2}{n\pi D_n}$$

其中：

$$D_n = (25 - n^2)^2 + (0.05n)^2$$

> **關鍵觀察**：當 $n^2 \approx 25$，即 $n \approx 5$ 時，$D_n$ 最小 → 該諧波的振幅最大（**共振現象 resonance**）。

### Step 3：疊加所有特解

$$y = y_1 + y_3 + y_5 + \cdots$$

每個諧波的振幅（amplitude）：

$$C_n = \sqrt{A_n^2 + B_n^2} = \frac{4}{n^2\pi\sqrt{D_n}}$$

### Step 4：數值結果

| $n$ | $C_n$ | 說明 |
|:---:|:---:|---|
| 1 | 0.0531 | 基頻，遠離共振 |
| 3 | 0.0088 | 偏離共振 |
| **5** | **0.2037** | **最接近共振（$n^2 = 25$），振幅最大** |
| 7 | 0.0011 | 高頻，衰減快 |
| 9 | 0.0003 | 高頻，振幅極小 |

> **系統的濾波效果（filtering）**：高頻諧波因 $1/n^2$ 衰減而幾乎消失，系統主要響應接近自然頻率 $\omega_0 = \sqrt{25} = 5$ 的諧波（$n = 5$）。輸出波形比輸入平滑。

---

## 4. 穩態響應的物理詮釋

### 4.1 共振（Resonance）

ODE $y'' + 0.05y' + 25y = r(t)$ 中：

- **自然頻率（natural frequency）**：$\omega_0 = \sqrt{k/m} = \sqrt{25} = 5$
- 驅動力的第 $n$ 個諧波頻率為 $n$
- 當 $n = 5$ 時，諧波頻率 $=$ 自然頻率 → **共振**
- 有阻尼（$c = 0.05 > 0$）時共振不會導致振幅無窮大，但仍最大

### 4.2 系統作為頻率濾波器（Frequency Filter）

```
輸入 r(t)（非正弦週期波）
    │
    ▼
Fourier 分解 → 各諧波成分
    │
    ▼
系統 ODE 對各頻率響應不同
    │
    ├─ 接近 ω₀ 的頻率 → 放大（amplify）
    └─ 遠離 ω₀ 的頻率 → 衰減（attenuate）
    │
    ▼
輸出 y(t)（比輸入更平滑）
```

這正是工程中**濾波器（filter）**的基本原理，在電路設計、音訊處理、控制系統中廣泛應用。

## 5. 求解流程總結

對任意週期驅動力 $r(t)$，完整求解流程如下：

**① 展開 $r(t)$**

$$r(t) = a_0 + \sum_{n=1}^{\infty}(a_n \cos nt + b_n \sin nt)$$

**② 對常數項 $a_0$ 求特解**

$$y_0 = \frac{a_0}{k}$$

（直接代入 ODE，$y_0'' = y_0' = 0$）

**③ 對每個諧波 $a_n\cos nt + b_n\sin nt$ 求特解**

設 $y_n = A_n\cos nt + B_n\sin nt$，代入 ODE 求係數：

| 係數 | 公式 |
|---|---|
| $A_n$ | $\displaystyle\frac{a_n(k - mn^2) + b_n cn}{(k - mn^2)^2 + c^2n^2}$ |
| $B_n$ | $\displaystyle\frac{b_n(k - mn^2) - a_n cn}{(k - mn^2)^2 + c^2n^2}$ |

**④ 疊加**

$$y_p = y_0 + \sum_{n=1}^{\infty} y_n = y_0 + \sum_{n=1}^{\infty}(A_n\cos nt + B_n\sin nt)$$

**⑤ 加上齊次解（暫態）**

$$y = y_p + y_h$$

若系統有阻尼（$c > 0$），$y_h \to 0$ 隨時間衰減，長期行為由 $y_p$ 決定。

## 重點整理

| 概念 | 內容 | 關鍵點 |
|---|---|---|
| 機械系統 ODE | $my'' + cy' + ky = r(t)$ | $r(t)$ 為任意週期力 |
| 電路系統 ODE | $LI'' + RI' + \frac{1}{C}I = E'(t)$ | 與機械系統完全類比 |
| 解題核心策略 | Fourier 分解 $r(t)$ → 對每個諧波求解 → 疊加 | 依賴線性 ODE 的疊加原理 |
| 特解形式 | $y_n = A_n\cos nt + B_n\sin nt$ | 待定係數法求 $A_n, B_n$ |
| 振幅 $C_n$ | $\sqrt{A_n^2 + B_n^2}$ | 反映系統對各頻率的響應強度 |
| 共振條件 | 諧波頻率 $\approx$ 自然頻率 $\omega_0 = \sqrt{k/m}$ | 有阻尼時振幅有限但最大 |
| 系統濾波效果 | 輸出比輸入平滑，高頻成分被衰減 | 應用於濾波器設計 |
| 暫態 vs. 穩態 | 有阻尼時 $y_h \to 0$，長期由 $y_p$ 主導 | 穩態響應 = Fourier 疊加的特解 |
