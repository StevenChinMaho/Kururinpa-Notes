# 2.7 Nonhomogeneous ODEs

## 基本結構

考慮 second-order nonhomogeneous linear ODE：

$$y'' + p(x)y' + q(x)y = r(x), \quad r(x) \neq 0$$

### General Solution 的組成

$$y(x) = y_h(x) + y_p(x)$$

| 符號 | 名稱 | 說明 |
|:---:|:---|:---|
| $y_h = c_1 y_1 + c_2 y_2$ | Homogeneous solution | 對應 $r(x) = 0$ 的 general solution |
| $y_p$ | Particular solution | 任意一個滿足原 nonhomogeneous ODE 的解，不含任意常數 |

> **物理意義**：$y_h$ 是系統的自然響應（transient，會隨時間衰減），$y_p$ 是外力驅動的穩態響應（steady state）。

---

## 定理一：解的疊加關係

**Theorem 1**

(a) nonhomogeneous ODE 的某個解，加上 homogeneous ODE 的某個解，仍是 nonhomogeneous ODE 的解。特別地，$y = y_h + y_p$ 是 nonhomogeneous ODE 的解。

(b) nonhomogeneous ODE 兩個解的**差**，是對應 homogeneous ODE 的解。

---

## 定理二：General Solution 涵蓋所有解

**Theorem 2**

若 $p(x)$、$q(x)$、$r(x)$ 在開區間 $I$ 上連續，則 nonhomogeneous ODE 在 $I$ 上的每一個解，都可以由 general solution $y = y_h + y_p$ 中令 $c_1$、$c_2$ 取適當值得到。

> Nonhomogeneous ODE 同樣**不存在 singular solutions**。

---

## 求解 $y_p$ 的方法一：Undetermined Coefficients（待定係數法）

### 適用條件

$r(x)$ 必須是以下形式，或其和與積：
- 指數函數 $e^{\gamma x}$
- $x$ 的冪次 $x^n$
- $\cos\omega x$ 或 $\sin\omega x$
- 上述的乘積（如 $e^{ax}\cos\omega x$）

### 查表規則

| $r(x)$ 的形式 | 猜測的 $y_p$ |
|:---:|:---:|
| $ke^{\gamma x}$ | $Ce^{\gamma x}$ |
| $kx^n \ (n = 0, 1, \ldots)$ | $K_n x^n + K_{n-1}x^{n-1} + \cdots + K_1 x + K_0$ |
| $k\cos\omega x$ 或 $k\sin\omega x$ | $K\cos\omega x + M\sin\omega x$ |
| $ke^{ax}\cos\omega x$ 或 $ke^{ax}\sin\omega x$ | $e^{ax}(K\cos\omega x + M\sin\omega x)$ |

### 三條選取規則

**(a) Basic Rule**：依上表選取對應的 $y_p$，代入 ODE 後比較係數求解。

**(b) Modification Rule**：若所猜測的 $y_p$ 中有某項剛好是 homogeneous ODE 的解，則將該項**乘以 $x$**（若對應的是 double root，則乘以 $x^2$）。

**(c) Sum Rule**：若 $r(x)$ 是多個函數的和，則 $y_p$ 取對應各項猜測的**和**。

---

## 範例

### 範例一：Basic Rule 應用

$$y'' + y = 0.001x^2, \quad y(0) = 0,\ y'(0) = 1.5$$

**Step 1** — Homogeneous solution：$y_h = A\cos x + B\sin x$

**Step 2** — 猜測 $y_p = K_2 x^2 + K_1 x + K_0$，代入得：

$$2K_2 + K_2 x^2 + K_1 x + K_0 = 0.001x^2$$

比較係數：$K_2 = 0.001$，$K_1 = 0$，$K_0 = -0.002$

$$y = A\cos x + B\sin x + 0.001x^2 - 0.002$$

**Step 3** — 代入 initial conditions：$A = 0.002$，$B = 1.5$

$$\boxed{y = 0.002\cos x + 1.5\sin x + 0.001x^2 - 0.002}$$

---

### 範例二：Modification Rule 應用

$$y'' + 3y' + 2.25y = -10e^{-1.5x}, \quad y(0) = 1,\ y'(0) = 0$$

**Step 1** — Characteristic equation：$(\lambda + 1.5)^2 = 0$，double root $\lambda = -1.5$

$$y_h = (c_1 + c_2 x)e^{-1.5x}$$

**Step 2** — $Ce^{-1.5x}$ 已是 homogeneous 解，依 Modification Rule 改猜 $y_p = Cx^2 e^{-1.5x}$，代入得 $C = -5$

$$y = (c_1 + c_2 x)e^{-1.5x} - 5x^2 e^{-1.5x}$$

**Step 3** — 代入 initial conditions：$c_1 = 1$，$c_2 = 1.5$

$$\boxed{y = (1 + 1.5x)e^{-1.5x} - 5x^2 e^{-1.5x}}$$

---

### 範例三：Sum Rule 應用

$$y'' + 2y' + 0.75y = 2\cos x - 0.25\sin x + 0.09x, \quad y(0) = 2.78,\ y'(0) = -0.43$$

**Step 1** — Characteristic equation：$(\lambda + \frac{1}{2})(\lambda + \frac{3}{2}) = 0$

$$y_h = c_1 e^{-x/2} + c_2 e^{-3x/2}$$

**Step 2** — 依 Sum Rule 猜測：

$$y_p = K\cos x + M\sin x + K_1 x + K_0$$

代入比較係數得：$K = 0$，$M = 1$，$K_1 = 0.12$，$K_0 = -0.32$

$$y = c_1 e^{-x/2} + c_2 e^{-3x/2} + \sin x + 0.12x - 0.32$$

**Step 3** — 代入 initial conditions：$c_1 = 3.1$，$c_2 = 0$

$$\boxed{y = 3.1e^{-x/2} + \sin x + 0.12x - 0.32}$$

---

## 求解 $y_p$ 的方法二：Variation of Parameters（參數變換法）

詳見 2.10 節。適用於 $r(x)$ 不在查表範圍內的情況（如 $\sec x$、$\ln x$）。

$$y_p(x) = -y_1 \int \frac{y_2 r}{W}\,dx + y_2 \int \frac{y_1 r}{W}\,dx$$

---

## 方法選擇策略

```
分析 r(x)
│
├─ r(x) = 0 ──→ Homogeneous method（特徵方程）
│
└─ r(x) ≠ 0
│
├─ r(x) 為指數、多項式、三角函數 ──→ Method 1: Undetermined Coefficients
│
└─ r(x) 為 sec x、ln x 等複雜形式 ──→ Method 2: Variation of Parameters
```

---

## 本節核心重點整理

| 概念 | 說明 |
|:---|:---|
| General solution | $y = y_h + y_p$，homogeneous 解加上任一 particular 解 |
| Undetermined Coefficients | 查表猜 $y_p$，適用於「好猜」的 $r(x)$ |
| Modification Rule | 猜測與 $y_h$ 重複時，乘以 $x$（或 $x^2$） |
| Sum Rule | $r(x)$ 為多項之和時，$y_p$ 也取各對應猜測之和 |
| Transient vs. Steady State | $y_h$ 衰減至零，$y_p$ 持續存在代表長期行為 |
