| 主題 | 說明 |
|---|---|
| 通解結構 | 齊次通解 + 特解 |

## 1. 非齊次系統的通解結構

$$\mathbf{y}' = \mathbf{A}(t)\mathbf{y} + \mathbf{g}(t)$$

**通解 = 齊次通解 + 特解：**

$$\boxed{\mathbf{y} = \mathbf{y}^{(h)} + \mathbf{y}^{(p)}}$$

| 部分 | 求法 | 說明 |
|---|---|---|
| $\mathbf{y}^{(h)}$（齊次通解） | 解 $\mathbf{y}' = \mathbf{A}\mathbf{y}$，求特徵值與特徵向量 | 含任意常數 $c_1, \ldots, c_n$ |
| $\mathbf{y}^{(p)}$（特解） | 待定係數法 或 參數變分法 | 不含任意常數 |

### 虛數情況

特徵值為 $\lambda = \alpha + \beta i$，特徵向量為 $x = a + bi$。

則兩個實數解必為：

1. $y_1(t) = e^{\alpha t} (a \cos \beta t - b \sin \beta t)$
    
2. $y_2(t) = e^{\alpha t} (a \sin \beta t + b \cos \beta t)$

---

## 2. 方法一：待定係數法（Method of Undetermined Coefficients）

### 2.1 適用條件

- **A** 的元素為**常數**
- **g**(t) 的各分量為以下類型：常數、$t$ 的正整數次冪、指數函數 $e^{\alpha t}$、$\cos\omega t$、$\sin\omega t$ 或其組合

**試探解的選取規則**（與 2.7 節單一 ODE 完全類似）：

| **g**(t) 的形式 | 試探解 $\mathbf{y}^{(p)}$ |
|---|---|
| 常數向量 | $\mathbf{a}$（常數向量） |
| $\mathbf{a}e^{\alpha t}$ | $\mathbf{u}e^{\alpha t}$ |
| $(\mathbf{a}_0 + \mathbf{a}_1 t + \cdots)e^{\alpha t}$ | $(\mathbf{u}_0 + \mathbf{u}_1 t + \cdots)e^{\alpha t}$ |
| $\mathbf{a}\cos\omega t + \mathbf{b}\sin\omega t$ | $\mathbf{u}\cos\omega t + \mathbf{v}\sin\omega t$ |

### 2.2 修正規則（Modification Rule）

> **若試探解中的指數 $\alpha$ 恰好是 A 的特徵值（即 $e^{\alpha t}$ 已出現在 $\mathbf{y}^{(h)}$ 中），必須對試探解乘以 $t$。**

具體而言，若 $\mathbf{g} = \mathbf{k}e^{\alpha t}$ 且 $\alpha = \lambda_i$（某特徵值），則試探解改為：

$$\mathbf{y}^{(p)} = \mathbf{u}te^{\alpha t} + \mathbf{v}e^{\alpha t}$$

而非 $\mathbf{u}e^{\alpha t}$（否則代入後無解）。

---

### 2.3 範例（Example 1）— 含修正規則

**題目：** 求通解

$$\mathbf{y}' = \underbrace{\begin{bmatrix}-3 & 1\\1 & -3\end{bmatrix}}_{\mathbf{A}}\mathbf{y} + \begin{bmatrix}-6\\2\end{bmatrix}e^{-2t}$$

**Step 1：解齊次系統**

由 4.3 節 Example 1 結果（$\lambda_1 = -2$，$\lambda_2 = -4$）：

$$\mathbf{y}^{(h)} = c_1\begin{bmatrix}1\\1\end{bmatrix}e^{-2t} + c_2\begin{bmatrix}1\\-1\end{bmatrix}e^{-4t}$$

**Step 2：判斷是否需要修正規則**

**g** 含 $e^{-2t}$，而 $\lambda_1 = -2$ 恰好是 **A** 的特徵值 → **需要修正規則**

試探解：

$$\mathbf{y}^{(p)} = \mathbf{u}te^{-2t} + \mathbf{v}e^{-2t}$$

**Step 3：求導並代入**

$$(\mathbf{y}^{(p)})' = \mathbf{u}e^{-2t} - 2\mathbf{u}te^{-2t} - 2\mathbf{v}e^{-2t} = \mathbf{A}\mathbf{u}te^{-2t} + \mathbf{A}\mathbf{v}e^{-2t} + \mathbf{g}$$

比較 $te^{-2t}$ 係數：$-2\mathbf{u} = \mathbf{A}\mathbf{u}$，即 $\mathbf{A}\mathbf{u} = -2\mathbf{u}$

→ **u** 為 $\lambda = -2$ 的特徵向量，取 $a = -2$：

$$\mathbf{u} = -2\begin{bmatrix}1\\1\end{bmatrix} = \begin{bmatrix}-2\\-2\end{bmatrix}$$

比較 $e^{-2t}$ 係數：$\mathbf{u} - 2\mathbf{v} = \mathbf{A}\mathbf{v} + \begin{bmatrix}-6\\2\end{bmatrix}$

即 $(\mathbf{A} - (-2)\mathbf{I})\mathbf{v} = \mathbf{u} - \begin{bmatrix}-6\\2\end{bmatrix}$，整理得：

$$v_1 - v_2 = -4 \implies \text{取 } v_1 = 0,\; v_2 = 4 \implies \mathbf{v} = \begin{bmatrix}0\\4\end{bmatrix}$$

**Step 4：組合通解**

$$\mathbf{y} = c_1\begin{bmatrix}1\\1\end{bmatrix}e^{-2t} + c_2\begin{bmatrix}1\\-1\end{bmatrix}e^{-4t} - 2\begin{bmatrix}1\\1\end{bmatrix}te^{-2t} + \begin{bmatrix}0\\4\end{bmatrix}e^{-2t}$$

## 3. 方法二：參數變分法（Method of Variation of Parameters）

### 3.1 適用條件

- 可用於**可變係數**矩陣 $\mathbf{A}(t)$ 或**任意形式** $\mathbf{g}(t)$
- 前提：**已知**齊次系統 $\mathbf{y}' = \mathbf{A}(t)\mathbf{y}$ 的基本解組

### 3.2 方法推導

**Step 1：建立基本矩陣**

設齊次系統的基底解為 $\mathbf{y}^{(1)}, \ldots, \mathbf{y}^{(n)}$，構成基本矩陣：

$$\mathbf{Y}(t) = \begin{bmatrix}\mathbf{y}^{(1)} & \cdots & \mathbf{y}^{(n)}\end{bmatrix}$$

齊次通解：$\mathbf{y}^{(h)} = \mathbf{Y}(t)\mathbf{c}$

**Step 2：將常數向量替換為函數**

令特解形式為：

$$\mathbf{y}^{(p)} = \mathbf{Y}(t)\mathbf{u}(t)$$

**Step 3：代入方程推導**

$$(\mathbf{y}^{(p)})' = \mathbf{Y}'\mathbf{u} + \mathbf{Y}\mathbf{u}' = \mathbf{A}\mathbf{Y}\mathbf{u} + \mathbf{g}$$

因 $\mathbf{Y}' = \mathbf{A}\mathbf{Y}$（基本矩陣性質），得：

$$\mathbf{Y}\mathbf{u}' = \mathbf{g} \implies \boxed{\mathbf{u}' = \mathbf{Y}^{-1}\mathbf{g}}$$

**Step 4：積分求 u**

$$\mathbf{u}(t) = \int \mathbf{Y}^{-1}(t)\,\mathbf{g}(t)\,dt$$

**Step 5：得特解**

$$\mathbf{y}^{(p)} = \mathbf{Y}(t)\mathbf{u}(t) = \mathbf{Y}(t)\int\mathbf{Y}^{-1}(t)\,\mathbf{g}(t)\,dt$$

### 3.3 完整公式

$$\boxed{\mathbf{y} = \mathbf{Y}(t)\mathbf{c} + \mathbf{Y}(t)\int\mathbf{Y}^{-1}(t)\,\mathbf{g}(t)\,dt}$$

---

### 3.4 範例（Example 2）— 參數變分法

**題目：** 與 Example 1 相同系統，改用參數變分法求解

$$\mathbf{y}' = \begin{bmatrix}-3 & 1\\1 & -3\end{bmatrix}\mathbf{y} + \begin{bmatrix}-6\\2\end{bmatrix}e^{-2t}$$

**Step 1：建立基本矩陣**

$$\mathbf{Y}(t) = \begin{bmatrix}e^{-2t} & e^{-4t}\\e^{-2t} & -e^{-4t}\end{bmatrix}$$

**Step 2：求 Y⁻¹**

$$\det(\mathbf{Y}) = e^{-2t}(-e^{-4t}) - e^{-4t}(e^{-2t}) = -2e^{-6t}$$

$$\mathbf{Y}^{-1} = \frac{1}{-2e^{-6t}}\begin{bmatrix}-e^{-4t} & -e^{-4t}\\-e^{-2t} & e^{-2t}\end{bmatrix} = \frac{1}{2}\begin{bmatrix}e^{2t} & e^{2t}\\e^{4t} & -e^{4t}\end{bmatrix}$$

**Step 3：計算 u'**

$$\mathbf{u}' = \mathbf{Y}^{-1}\mathbf{g} = \frac{1}{2}\begin{bmatrix}e^{2t} & e^{2t}\\e^{4t} & -e^{4t}\end{bmatrix}\begin{bmatrix}-6\\2\end{bmatrix}e^{-2t} = \begin{bmatrix}-2\\-4e^{2t}\end{bmatrix}$$

**Step 4：積分求 u**

$$\mathbf{u}(t) = \int_0^t\begin{bmatrix}-2\\-4e^{2\tilde{t}}\end{bmatrix}d\tilde{t} = \begin{bmatrix}-2t\\-2e^{2t}+2\end{bmatrix}$$

**Step 5：得特解**

$$\mathbf{y}^{(p)} = \mathbf{Y}(t)\mathbf{u}(t) = \begin{bmatrix}e^{-2t} & e^{-4t}\\e^{-2t} & -e^{-4t}\end{bmatrix}\begin{bmatrix}-2t\\-2e^{2t}+2\end{bmatrix}$$

$$= \begin{bmatrix}-2t-2\\-2t+2\end{bmatrix}e^{-2t} + \begin{bmatrix}2\\-2\end{bmatrix}e^{-4t}$$

**Step 6：通解**

$$\mathbf{y} = c_1\begin{bmatrix}1\\1\end{bmatrix}e^{-2t} + c_2\begin{bmatrix}1\\-1\end{bmatrix}e^{-4t} - 2\begin{bmatrix}1\\1\end{bmatrix}te^{-2t} + \begin{bmatrix}-2\\2\end{bmatrix}e^{-2t}$$

> **注意：** Example 1 與 Example 2 的特解差了一個 $\begin{bmatrix}-2\\2\end{bmatrix}e^{-2t}$ 項，但這個差值恰好是齊次解的一部分（可被 $c_1$ 吸收），兩個答案本質相同。

## 4. 兩種方法的比較

| 比較項目 | 待定係數法 | 參數變分法 |
|---|---|---|
| **適用範圍** | **A** 為常數矩陣，**g** 為特定函數類型 | **A** 可為 $t$ 的函數，**g** 為任意連續函數 |
| **計算複雜度** | 較簡單（直接設試探解） | 較複雜（需求 **Y**⁻¹ 並積分） |
| **修正規則** | 需注意（$\alpha = \lambda_i$ 時乘以 $t$） | 不需要（自動處理共振情況） |
| **普適性** | 有限制 | 完全普適 |
| **最佳使用時機** | **g** 結構簡單，且確認無修正規則問題 | **g** 複雜、**A** 含 $t$、或有多個特徵值衝突 |

## 5. 解題流程總結

```
非齊次系統 y' = Ay + g
        │
Step 1：解齊次系統 y' = Ay
        → 求特徵值 λ、特徵向量 x
        → 建立齊次通解 y⁽ʰ⁾ = Y(t)c
        │
Step 2：選擇求特解的方法
        │
        ├── g 為常數/冪次/指數/三角函數，A 為常數
        │   → 待定係數法
        │       ↓
        │   檢查：g 中的指數是否為 A 的特徵值？
        │       YES → 試探解乘以 t（修正規則）
        │       NO  → 直接代入試探解求係數
        │
        └── 其他情況（A 含 t，g 複雜）
            → 參數變分法
                ↓
            計算 Y⁻¹，積分 u' = Y⁻¹g
            特解 y⁽ᵖ⁾ = Y(t)u(t)
        │
Step 3：通解 y = y⁽ʰ⁾ + y⁽ᵖ⁾
        │
Step 4：若有初始條件，代入求 c₁, c₂, …
```

## 6. 與第二章方法的對應關係

本節方法是第二章單一 ODE 方法的**矩陣推廣**：

| 第二章（單一 ODE） | 第四章（ODE 系統） |
|---|---|
| $y'' + py' + qy = g(t)$ | $\mathbf{y}' = \mathbf{A}\mathbf{y} + \mathbf{g}(t)$ |
| 特徵方程式 → 特徵根 | 特徵值問題 → 特徵向量 |
| 齊次通解 + 特解 | 齊次通解 + 特解（結構完全相同） |
| 待定係數法（2.7 節） | 待定係數法（**含修正規則**） |
| 參數變分法（2.10 節） | 參數變分法（$\mathbf{u}' = \mathbf{Y}^{-1}\mathbf{g}$） |

> **修正規則的差異：** 第二章的修正規則針對特徵根重複時乘以 $t$；第四章的修正規則針對 **g** 中的指數恰好等於矩陣 **A** 某個特徵值時使用，邏輯相同，形式為矩陣版本。

## 重點整理

| 概念 | 關鍵內容 |
|---|---|
| 通解結構 | $\mathbf{y} = \mathbf{y}^{(h)} + \mathbf{y}^{(p)}$，齊次通解加特解 |
| 待定係數法適用 | **A** 常數，**g** 為常數/冪次/指數/三角 |
| 修正規則 | **g** 中指數 = **A** 的特徵值時，試探解改為 $\mathbf{u}te^{\alpha t} + \mathbf{v}e^{\alpha t}$ |
| 參數變分法核心 | $\mathbf{u}' = \mathbf{Y}^{-1}\mathbf{g}$，特解 $= \mathbf{Y}\mathbf{u}$ |
| 基本矩陣 **Y** | 由齊次基底解排列成行，滿足 $\mathbf{Y}' = \mathbf{A}\mathbf{Y}$ |
| 兩法的特解差異 | 可差一個齊次解項（被 $c_i$ 吸收），通解等價 |
