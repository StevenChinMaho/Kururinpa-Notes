## 1. 引入：從線性代數的基底到函數空間

在線性代數中，平面向量 $\mathbf{v}$ 可以用基底 $\mathbf{e}_1, \mathbf{e}_2$ 的線性組合表示：

$$\mathbf{v} = a_1 \mathbf{e}_1 + a_2 \mathbf{e}_2$$

若 $\mathbf{e}_1, \mathbf{e}_2$ 為**正規正交基底（orthonormal basis）**，係數可由內積直接求得：

$$\mathbf{v} = \langle \mathbf{v}, \mathbf{e}_1 \rangle \mathbf{e}_1 + \langle \mathbf{v}, \mathbf{e}_2 \rangle \mathbf{e}_2$$

> **類比關鍵**：Fourier Analysis 把這個想法延伸到**函數空間**。每個「基底向量」換成一個 sin 或 cos 函數，且有**無窮多個**基底。「係數」則由積分（內積的函數版）求得。

$$f(x) = \langle f, e_1 \rangle e_1 + \langle f, e_2 \rangle e_2 + \cdots$$

對應到 Fourier Series 的形式：

$$f(x) = a_0 + \sum_{n=1}^{\infty} \left( a_n \cos\frac{n\pi x}{L} + b_n \sin\frac{n\pi x}{L} \right)$$

## 2. Fourier Series：定義與係數

### 2.1 週期函數（Periodic Function）

函數 $f(x)$ 若滿足：

$$f(x + p) = f(x) \quad \text{對所有 } x$$

則稱 $f(x)$ 為**週期為 $p$ 的週期函數**。

### 2.2 Fourier Series（週期 $2\pi$）

對週期為 $2\pi$ 的函數 $f(x)$，其 Fourier Series 為：

$$f(x) = a_0 + \sum_{n=1}^{\infty} \left( a_n \cos nx + b_n \sin nx \right)$$

其中基底函數系為：

$$1,\; \cos x,\; \sin x,\; \cos 2x,\; \sin 2x,\; \ldots,\; \cos nx,\; \sin nx,\; \ldots$$

### 2.3 Euler 公式（Fourier Coefficients）

Fourier 係數由以下積分（Euler formulas）計算：

$$\boxed{a_0 = \frac{1}{2\pi} \int_{-\pi}^{\pi} f(x)\, dx}$$

$$\boxed{a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \cos nx\, dx \qquad n = 1, 2, \ldots}$$

$$\boxed{b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \sin nx\, dx \qquad n = 1, 2, \ldots}$$

> $a_0$ 是 $f(x)$ 在一個週期內的**平均值**（乘上 $\pi$ 之前）。

---

## 3. 正交性（Orthogonality of the Trigonometric System）

三角函數系在區間 $[-\pi, \pi]$ 上具有**正交性**：任意兩個不同基底函數之乘積，在一個週期上的積分為零。具體而言，對任意整數 $n, m$：

| 公式 | 條件 | 結果 |
|---|---|---|
| $\displaystyle\int_{-\pi}^{\pi} \cos nx \cos mx\, dx$ | $n \neq m$ | $0$ |
| $\displaystyle\int_{-\pi}^{\pi} \sin nx \sin mx\, dx$ | $n \neq m$ | $0$ |
| $\displaystyle\int_{-\pi}^{\pi} \sin nx \cos mx\, dx$ | $n \neq m$ 或 $n = m$ | $0$ |

> 正交性是 Euler 公式成立的根本原因：用 $\cos mx$ 乘以 Fourier Series 兩側再積分，其他項因正交性全部消零，只剩 $a_m$ 項。

---

## 4. 收斂定理（Convergence Theorem）

**定理**（Representation by a Fourier Series）：

設 $f(x)$ 為週期 $2\pi$ 的函數，在 $[-\pi, \pi]$ 上**逐段連續（piecewise continuous）**，且在每一點都有**左微分與右微分**，則 $f(x)$ 的 Fourier Series 收斂，且：

- 在 $f(x)$ 的**連續點**：級數的和 $= f(x)$
- 在 $f(x)$ 的**不連續點 $x_0$**：級數的和 $=$ **左極限與右極限的平均值**

$$S(x_0) = \frac{f(x_0 - 0) + f(x_0 + 0)}{2}$$

其中 $f(x_0 - 0)$ 為左極限，$f(x_0 + 0)$ 為右極限。

## 5. Example 1：週期矩形波（Periodic Rectangular Wave）

### 問題

$$f(x) = \begin{cases} -k & \text{if } -\pi < x < 0 \\ k & \text{if } 0 < x < \pi \end{cases}, \quad f(x + 2\pi) = f(x)$$

### 計算 $a_n$

$$a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \cos nx\, dx = \frac{1}{\pi} \left[ \int_{-\pi}^{0} (-k) \cos nx\, dx + \int_{0}^{\pi} k \cos nx\, dx \right]$$

$$= \frac{1}{\pi} \left[ -k \frac{\sin nx}{n} \Bigg|_{-\pi}^{0} + k \frac{\sin nx}{n} \Bigg|_{0}^{\pi} \right] = 0$$

所有 $a_n = 0$（包含 $a_0 = 0$），因為 $f(x)$ 為**奇函數**，cos 為偶函數，乘積為奇函數，在對稱區間積分為零。

### 計算 $b_n$

$$b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \sin nx\, dx = \frac{2k}{n\pi}(1 - \cos n\pi)$$

利用 $\cos n\pi = (-1)^n$：

$$1 - \cos n\pi = \begin{cases} 2 & n \text{ 為奇數} \\ 0 & n \text{ 為偶數} \end{cases}$$

因此：

$$b_1 = \frac{4k}{\pi}, \quad b_2 = 0, \quad b_3 = \frac{4k}{3\pi}, \quad b_4 = 0, \quad b_5 = \frac{4k}{5\pi}, \quad \ldots$$

### 結果：Fourier Series

$$f(x) = \frac{4k}{\pi} \left( \sin x + \frac{1}{3}\sin 3x + \frac{1}{5}\sin 5x + \cdots \right)$$

### 特定點的代入（推導 $\pi$ 的著名公式）

在 $x = \pi/2$ 代入（此為連續點，$f(\pi/2) = k$）：

$$k = \frac{4k}{\pi} \left(1 - \frac{1}{3} + \frac{1}{5} - + \cdots \right)$$

$$\therefore \quad \boxed{1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + - \cdots = \frac{\pi}{4}}$$

### Partial Sums（部分和）的收斂過程

| 部分和 | 包含的項 |
|---|---|
| $S_1$ | $\frac{4k}{\pi} \sin x$ |
| $S_2$ | $S_1 + \frac{4k}{3\pi} \sin 3x$ |
| $S_3$ | $S_2 + \frac{4k}{5\pi} \sin 5x$ |

加入越多項，曲線越接近矩形波，但在不連續點（$x = 0, \pm\pi$）附近始終存在**過衝（overshoot）**，此現象稱為 **Gibbs phenomenon**（在 11.4 節討論）。

## 重點整理

| 概念 / 公式 | 內容 | 關鍵注意事項 |
|---|---|---|
| Fourier Series | $\displaystyle f(x) = a_0 + \sum_{n=1}^{\infty}(a_n \cos nx + b_n \sin nx)$ | 僅適用於週期 $2\pi$ 函數；一般週期見 11.2 |
| $a_0$（常數項） | $\displaystyle\frac{1}{2\pi}\int_{-\pi}^{\pi} f(x)\,dx$ | 等於函數一個週期的平均值 |
| $a_n$（cos 係數） | $\displaystyle\frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\cos nx\,dx$ | 若 $f$ 為奇函數，$a_n = 0$ |
| $b_n$（sin 係數） | $\displaystyle\frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\sin nx\,dx$ | 若 $f$ 為偶函數，$b_n = 0$ |
| 正交性 | 不同基底函數乘積在 $[-\pi,\pi]$ 積分 $= 0$ | 這是 Euler 公式成立的基礎 |
| 收斂（連續點） | Fourier Series 之和 $= f(x)$ | 需滿足逐段連續＋左右微分存在 |
| 收斂（不連續點 $x_0$） | $\displaystyle\frac{f(x_0^-) + f(x_0^+)}{2}$ | 收斂到左右極限的平均值 |
| 奇函數特性 | 矩形波為奇函數 → $a_n = 0$，只有 sin 項 | 對稱性可大幅簡化計算（詳見 11.2） |
