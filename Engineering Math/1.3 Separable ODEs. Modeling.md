# 1.3 Separable ODEs. Modeling

## 1. 可分離變數常微分方程式 (Separable ODEs) 的基本解法

- **標準形式**：$g(y)y' = f(x)$ 。
    
- **求解步驟**：將方程式改寫為 $g(y)dy = f(x)dx$，接著對兩邊進行積分，即可得到通解 $\int g(y)dy = \int f(x)dx + c$ 。
    
- **範例 (鐘形曲線 Bell-Shaped Curve)**：求解初始值問題 $y' = -2xy$ 且 $y(0) = 1.8$ 。移項後積分得到 $\frac{dy}{y} = -2x dx$，推導出 $\ln y = -x^2 + c_1$，通解為 $y = ce^{-x^2}$ 。代入初始條件可求得特解 $y = 1.8e^{-x^2}$ 。
    

## 2. 可化為可分離形式的延伸方法 (Reduction to Separable Form)

- 當遇到無法直接分離變數的 ODE，但其形式可以寫為 $y' = f(\frac{y}{x})$ 時（其中 $f$ 是 $\frac{y}{x}$ 的任意函數），可以透過變數變換來處理 。
    
- 令 $u = \frac{y}{x}$，則 $y = ux$；利用乘法微分公式可得 $y' = u'x + u$ 。
    
- 將其代回原式後可得 $u'x + u = f(u)$，進一步移項即可轉換為可分離變數的型態：$\frac{du}{f(u)-u} = \frac{dx}{x}$ 。
    
- **範例**：解方程式 $2xyy' = y^2 - x^2$ 。透過令 $u = \frac{y}{x}$ 進行轉換，最終可解得通解 $x^2 + y^2 = cx$ 。此通解在幾何上代表一個通過原點、且圓心位在 x 軸上的圓形族 。
    

## 3. 物理與實際應用的建模範例

本節透過多個實際案例展示如何利用可分離變數 ODE 進行數學建模與求解：

- **放射性碳定年法 (Radiocarbon Dating)**：
    
    - 在生物體活著時，體內放射性碳-14與一般碳-12的比例會保持恆定；生物死亡後，吸收終止，碳-14便開始衰變 。
        
    - 藉由比較化石與大氣中放射性碳的比例，可以估算出化石的年齡 。其衰變模型為 $y' = ky$，解為 $y = y_0e^{kt}$ 。
        
    - 已知碳-14的半衰期為 5715 年，可算出衰變常數 $k = -0.0001213$ 。若測得「冰人奧茨 (Oetzi)」體內的碳-14比例為活體的 52.5% ($0.525 = e^{-0.0001213t}$)，推算其距今約 5312 年前 。
        
- **牛頓冷卻定律 (Newton’s Law of Cooling)**：
    
    - 實驗顯示，導熱良好物體的溫度 $T$ 隨時間的變化率，會與物體溫度和周遭環境溫度 $T_A$ 之間的溫差成正比 。
        
    - 該物理過程的模型為 $\frac{dT}{dt} = k(T - T_A)$ ，可透過分離變數法解出溫度變化函數 。
        
- **腫瘤的貢珀茨生長模型 (Gompertz growth in tumors)**：
    
    - 用以描述腫瘤細胞質量的模型為 $y' = -Ay \ln y$ (其中 $A>0$) 。
        
    - 分析方程式可知，當 $y<1$ 時腫瘤成長 ($y'>0$)；當 $y>1$ 時腫瘤衰退 ($y'<0$) 。這與腫瘤內部細胞因缺氧或養分不足而死亡的臨床觀察相符 。
        
    - 此模型的常數解 (Constant solution) 為 $y(t)=1$ 。
        
- **繫船繩的摩擦力 (Rope)**：
    
    - 實驗顯示，繩索微小部分的受力變化 $\Delta S$ 與其張力 $S$ 以及微小的纏繞角度 $\Delta\phi$ 成正比 。
        
    - 若比例常數為 0.15，模型即為 $\frac{dS}{d\phi} = 0.15S$，解出 $S = S_0e^{0.15\phi}$ 。若要讓岸上的人能抵抗船隻 1000 倍的拉力 ($S = 1000S_0$)，計算結果顯示大約需要將繩子繞著繫船柱 8 圈 。
