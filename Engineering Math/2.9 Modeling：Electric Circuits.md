# 2.9 Modeling: Electric Circuits

## 電路模型概覽

本節將 second-order ODE 應用於 **RLC 串聯電路**，透過 Kirchhoff's Voltage Law (KVL) 建立電路的數學模型，與機械振動系統形成完美的對應關係。

---

## RLC 串聯電路

電路由三個元件串聯組成，接上交流電源 $E(t) = E_0 \sin\omega t$：

|元件|符號|電壓降（Voltage Drop）|
|:-:|:-:|:-:|
|電阻 Resistor|$R$|$RI$|
|電感 Inductor|$L$|$L\dfrac{dI}{dt}$|
|電容 Capacitor|$C$|$\dfrac{Q}{C} = \dfrac{1}{C}\displaystyle\int I\,dt$|

其中 $I(t)$ 為電流，$Q(t)$ 為電荷量，兩者關係為 $I = Q'$。

---

## 數學模型建立

### KVL 方程式

![[2.9.1.png]]

對 RLC 電路套用 KVL（迴路電壓總和為零）：

$$LI' + RI + \frac{1}{C}\int I,dt = E(t) = E_0\sin\omega t$$

### 對時間微分（以電流 $I$ 表示）

$$\boxed{LI'' + RI' + \frac{1}{C}I = E'(t) = E_0\omega\cos\omega t}$$

### 以電荷 $Q$ 表示

$$\boxed{LQ'' + RQ' + \frac{1}{C}Q = E(t)}$$

> **與機械系統的對應**：此方程式與彈簧-質量-阻尼系統 $mx'' + cx' + kx = F(t)$ 在數學結構上完全相同。

### 類比對照表

|電路量|機械量|
|:-:|:-:|
|電感 $L$|質量 $m$|
|電阻 $R$|阻尼係數 $c$|
|$1/C$（電容倒數）|彈簧係數 $k$|
|電壓 $E(t)$|外力 $F(t)$|
|電流 $I(t)$|速度 $x'(t)$|
|電荷 $Q(t)$|位移 $x(t)$|

---

## 解的結構

General solution 同樣分為兩部分：

$$I(t) = I_h(t) + I_p(t)$$

|部分|名稱|說明|
|:--|:-:|:--|
|$I_h$|Transient current|暫態電流，依 $R$、$L$、$C$ 指數衰減至零|
|$I_p$|Steady-state current|穩態電流，跟隨電源頻率持續振盪|

---

## 範例：RLC 電路求解

**題目**：$R = 11,\Omega$，$L = 0.1,\text{H}$，$C = 10^{-2},\text{F}$， 電源 $E(t) = 110\sin 377t$（60 Hz）。初始條件：$I(0) = 0$，$Q(0) = 0$。

### Step 1：Homogeneous ODE 的 General Solution

ODE 整理為：

$$0.1I'' + 11I' + 100I = 110 \times 377\cos 377t$$

Characteristic equation：

$$0.1\lambda^2 + 11\lambda + 100 = 0.1(\lambda + 10)(\lambda + 100) = 0$$

根為 $\lambda_1 = -10$，$\lambda_2 = -100$（兩個相異實根，Case I）

$$I_h = c_1 e^{-10t} + c_2 e^{-100t}$$

### Step 2：Particular Solution $I_p$

$r(t) = E_0\omega\cos\omega t$ 為餘弦函數，依 undetermined coefficients 猜測：

$$I_p = a\cos 377t + b\sin 377t$$

代入求解（其中 $\omega = 377$，$R = 11$，reactance $= \omega L - 1/(\omega C) = 37.4$）：

$$a = \frac{-110 \times 37.4}{11^2 + 37.4^2} = -2.71, \quad b = \frac{110 \times 11}{11^2 + 37.4^2} = 0.796$$

$$I(t) = c_1 e^{-10t} + c_2 e^{-100t} - 2.71\cos 377t + 0.796\sin 377t$$

### Step 3：代入 Initial Conditions

由 $I(0) = 0$：

$$c_1 + c_2 - 2.71 = 0 \quad \cdots (1)$$

由 $Q(0) = 0$，在 $t = 0$ 代入原始 KVL 方程式 $LI'(0) + RI(0) + Q(0)/C = E(0)$， 得 $I'(0) = 0$，因此：

$$I'(0) = -10c_1 - 100c_2 + 0.796 \times 377 = 0 \quad \cdots (2)$$

解聯立：$c_1 = -0.323$，$c_2 = 3.033$

$$\boxed{I(t) = -0.323e^{-10t} + 3.033e^{-100t} - 2.71\cos 377t + 0.796\sin 377t}$$

> 暫態項 $I_h$ 在數秒內衰減消失，剩下穩態電流 $I_p = -2.71\cos 377t + 0.796\sin 377t$ 持續以 60 Hz 振盪。

---

## 求解流程總結

```
建立 KVL 方程式
       │
       ▼
  整理成標準 ODE 形式
  LI'' + RI' + (1/C)I = E'(t)
       │
       ▼
Step 1：解 homogeneous ODE → I_h
  （求 characteristic equation 的根）
       │
       ▼
Step 2：求 particular solution I_p
  （用 undetermined coefficients）
       │
       ▼
Step 3：I = I_h + I_p
  代入 initial conditions 求 c₁, c₂
```

## 本節核心重點整理

|概念|說明|
|:--|:--|
|**KVL**|迴路電壓降之和等於電源電壓|
|**三個電壓降**|$RI$（電阻）、$LI'$（電感）、$Q/C$（電容）|
|**ODE 形式**|$LI'' + RI' + \frac{1}{C}I = E'(t)$，標準 nonhomogeneous 二階線性 ODE|
|**Transient**|$I_h$，由 $R$、$L$、$C$ 決定，指數衰減|
|**Steady-state**|$I_p$，跟隨電源頻率，長期行為|
|**電路-機械類比**|$L \leftrightarrow m$，$R \leftrightarrow c$，$1/C \leftrightarrow k$|
