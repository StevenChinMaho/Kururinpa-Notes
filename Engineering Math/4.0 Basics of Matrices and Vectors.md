## 1. 特徵值問題（Eigenvalue Problem）

### 1.1 定義

對於 n × n 矩陣 **A**，若存在非零向量 **x** 與純量 λ 滿足：

$$\mathbf{A}\mathbf{x} = \lambda\mathbf{x}$$

則 λ 稱為**特徵值（eigenvalue）**，**x** 稱為對應的**特徵向量（eigenvector）**。

> 特徵值是系統的「DNA」：它的正負與虛實決定了系統是衰減、爆炸還是振盪。

### 1.2 求解步驟

**Step 1：建立特徵方程式**

$$\det(\mathbf{A} - \lambda\mathbf{I}) = 0$$

**Step 2：求解 λ**（解二次（或 n 次）多項式）

**Step 3：代回求特徵向量**

對每個 λ，求解齊次線性方程組 $(\mathbf{A} - \lambda\mathbf{I})\mathbf{x} = \mathbf{0}$，找出非零解 **x**。

---

## 2. 2×2 矩陣的特徵方程式

對 $\mathbf{A} = \begin{bmatrix} a_{11} & a_{12} \\ a_{21} & a_{22} \end{bmatrix}$，特徵方程式展開為：

$$\lambda^2 - p\lambda + q = 0$$

其中定義三個關鍵量：

| 符號 | 名稱 | 定義 | 意義 |
|---|---|---|---|
| $p$ | 跡數（Trace） | $p = a_{11} + a_{22} = \lambda_1 + \lambda_2$ | 特徵值之和 |
| $q$ | 行列式（Determinant） | $q = \det\mathbf{A} = \lambda_1 \lambda_2$ | 特徵值之積 |
| $\Delta$ | 判別式（Discriminant） | $\Delta = p^2 - 4q = (\lambda_1 - \lambda_2)^2$ | 特徵值的「距離平方」 |

透過 p 和 q 即可**不求解特徵值**就預測系統行為（詳見 4.4 節）。

---

## 3. 範例解析（Example 1）

**題目：** 求矩陣 $\mathbf{A} = \begin{bmatrix} -4.0 & 4.0 \\ -1.6 & 1.2 \end{bmatrix}$ 的特徵值與特徵向量。

### Step 1：建立特徵方程式

$$\det(\mathbf{A} - \lambda\mathbf{I}) = \begin{vmatrix} -4-\lambda & 4 \\ -1.6 & 1.2-\lambda \end{vmatrix} = \lambda^2 + 2.8\lambda + 1.6 = 0$$

### Step 2：求特徵值

$$\lambda_1 = -2, \quad \lambda_2 = -0.8$$

兩個特徵值均為**實數且為負數** → 系統穩定衰減。

### Step 3：求特徵向量

**對 λ₁ = −2：**

代入 $(\mathbf{A} - \lambda_1\mathbf{I})\mathbf{x} = \mathbf{0}$：

$$(-4.0 + 2.0)x_1 + 4.0x_2 = 0 \implies -2x_1 + 4x_2 = 0 \implies x_1 = 2x_2$$

取 $x_2 = 1$ → $\mathbf{x}^{(1)} = \begin{bmatrix} 2 \\ 1 \end{bmatrix}$

**對 λ₂ = −0.8：**

同理可得 $\mathbf{x}^{(2)} = \begin{bmatrix} 1 \\ 0.8 \end{bmatrix}$

## 4. 特徵值的幾何意義

```
矩陣 A
   ↓  det(A − λI) = 0
特徵方程式 λ² − pλ + q = 0
   ↓  解方程
特徵值 λ₁, λ₂
   ↓  決定
幾何命運（Geometric Destiny）：衰減 / 爆炸 / 振盪
```

| 特徵值特性 | 系統行為 |
|---|---|
| 實數，皆為負 | 穩定衰減（trajectories → 原點） |
| 實數，皆為正 | 不穩定發散（trajectories → ∞） |
| 實數，異號 | 鞍點（saddle point），不穩定 |
| 純虛數 | 中心（center），永久振盪 |
| 複數（實部 < 0） | 穩定螺旋（spiral inward） |
| 複數（實部 > 0） | 不穩定螺旋（spiral outward） |

## 重點整理

| 概念 | 公式 / 方法 | 關鍵注意事項 |
|---|---|---|
| 特徵值 | $\det(\mathbf{A} - \lambda\mathbf{I}) = 0$ | 解 n 次多項式，n×n 矩陣有 n 個根（含重根與複數根） |
| 特徵向量 | $(\mathbf{A} - \lambda\mathbf{I})\mathbf{x} = \mathbf{0}$ 的非零解 | 特徵向量不唯一，任意純量倍數均為特徵向量 |
| 跡數 p | $p = a_{11} + a_{22}$ | $p = \lambda_1 + \lambda_2$ |
| 行列式 q | $q = \det\mathbf{A}$ | $q = \lambda_1 \lambda_2$；q < 0 表示異號特徵值（鞍點） |
| 判別式 Δ | $\Delta = p^2 - 4q$ | Δ > 0：實數不同根；Δ = 0：重根；Δ < 0：複數根 |
