# 1.5 Linear ODEs. Bernoulli equation. Population dynamics

## 1. 線性常微分方程式 (Linear ODEs)

- **基本定義**：如果一個一階 ODE 能夠透過代數方法整理成 $y' + p(x)y = r(x)$ 的形式，則稱為線性常微分方程式 。方程式右側的 $r(x)$ 在工程上常被稱為「輸入 (input)」，而解 $y(x)$ 則稱為「輸出 (output)」或對輸入的「響應 (response)」 。
    
- **齊次線性常微分方程式 (Homogeneous Linear ODE)**：
    
    - 當 $r(x) = 0$ 時，方程式變為 $y' + p(x)y = 0$，稱為齊次方程式 。
        
    - 透過分離變數法，可得到齊次 ODE 的通解為 $y(x) = ce^{-\int p(x)dx}$ 。
        
- **非齊次線性常微分方程式 (Nonhomogeneous Linear ODE)**：
    
    - 當 $r(x) \neq 0$ 時，方程式為非齊次 。
        
    - 求解時需使用積分因子 (Integrating factor) $F(x) = e^h$，其中 $h = \int p(x)dx$ 。
        
    - 最終通解公式為 $y(x) = e^{-h}(\int e^hr dx + c)$ 。解的結構可以理解為「總輸出 = 對輸入 $r$ 的響應 + 對初始數據的響應」 。
        

## 2. 白努利方程式 (Bernoulli Equation) 與化為線性形式

- **方程式形式**：白努利方程式的形式為 $y' + p(x)y = g(x)y^a$，其中 $a$ 可以是任意實數 。
    
- **求解技巧 (變數變換)**：
    
    - 它不是線性的，但可以透過設定新變數 $u(x) = [y(x)]^{1-a}$ 來將其轉換為線性形式 。
        
    - 微分後代回，可得到一個關於 $u$ 的線性 ODE：$u' + (1-a)pu = (1-a)g$ 。求出 $u$ 後再轉換回 $y$ 即可解題。
        

## 3. 自主常微分方程式 (Autonomous ODE) 與穩定性

- **自主 ODE 定義**：如果方程式中沒有明確出現自變數 $t$，即形式為 $y' = f(y)$，則稱為自主常微分方程式 。
    
- **平衡解 (Equilibrium Solutions) 與臨界點 (Critical Points)**：
    
    - 滿足 $f(y) = 0$ 的常數解稱為平衡解 。使函數值為零的點即為臨界點 。
        
- **穩定性判定**：
    
    - **穩定 (Stable)**：如果初始值靠近平衡解的曲線，隨著時間 $t$ 增加，仍持續靠近該平衡解，則稱其為穩定 。
        
    - **不穩定 (Unstable)**：如果初始值靠近平衡解的曲線，隨著時間 $t$ 增加反而逐漸遠離，則稱其為不穩定 。
        

## 4. 物理與族群動態建模範例

- **電路學 (RL 電路)**：利用克希荷夫電壓定律 (KVL) 可建立模型 $LI' + RI = E(t)$ ，這是一個標準的線性 ODE，可透過積分因子求出電流 $I(t)$ 的通解 。
    
- **族群動態與羅吉斯方程式 (Logistic Equation)**：
	- 模型形式為 $y' = Ay - By^2$。
	    
	- 當 $B=0$ 時代表族群呈指數增長。項次 $-By^2$ 是一個「煞車項 (braking term)」，用以防止族群無限制增長。
	    
	- 當族群數量小於極限值 $A/B$ 時會持續增長 ($y'>0$)；大於 $A/B$ 時則會減少 ($y'<0$)，最終極限會趨近於 $A/B$。

- **傳染病蔓延模型 (Epidemics)**：
    
    - 模型為 $y' = ky(1-y)$ 。
        
    - 具有兩個平衡解：$y_1 = 0$（不穩定，因為稍微有人感染就會讓 $y'>0$ 導致疫情擴散）以及 $y_2 = 1$（穩定，最終比例會趨近於 1，代表所有人都被感染） 。
