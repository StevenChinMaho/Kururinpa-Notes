## 1. 非線性自治系統（Nonlinear Autonomous System）

### 1.1 系統形式

$$\mathbf{y}' = \mathbf{f}(\mathbf{y}), \quad \text{即} \begin{cases} y_1' = f_1(y_1, y_2) \\ y_2' = f_2(y_1, y_2) \end{cases}$$

**自治（autonomous）** 的意義：右側函數 $f_1, f_2$ **不顯含**獨立變數 $t$，系統行為由當前狀態完全決定。

### 1.2 臨界點的求法

臨界點為 $\mathbf{y}' = \mathbf{0}$ 的解，即同時滿足：

$$f_1(y_1, y_2) = 0 \quad \text{且} \quad f_2(y_1, y_2) = 0$$

### 1.3 臨界點的平移

若臨界點 $P_0$ 不在原點 $(a, b)$，為方便分析，令：

$$\tilde{y}_1 = y_1 - a, \quad \tilde{y}_2 = y_2 - b$$

將 $P_0$ 平移至原點後再進行線性化。

---

## 2. 線性化定理（Theorem 1 — Linearization）

### 2.1 核心思想

非線性系統在臨界點附近的行為，可用其**線性近似系統**來描述——就像用放大鏡看曲面，局部看起來是平面。

### 2.2 線性化的形式

將非線性系統寫成：

$$\mathbf{y}' = \mathbf{A}\mathbf{y} + \mathbf{h}(\mathbf{y})$$

其中 $\mathbf{A}$ 為常數矩陣（線性部分），$\mathbf{h}(\mathbf{y})$ 為高階非線性餘項。

**線性化矩陣 A 即為 Jacobian 矩陣在臨界點的值：**

$$\mathbf{A} = \mathbf{J}\big|_{P_0} = \begin{bmatrix} \dfrac{\partial f_1}{\partial y_1} & \dfrac{\partial f_1}{\partial y_2} \\[10pt] \dfrac{\partial f_2}{\partial y_1} & \dfrac{\partial f_2}{\partial y_2} \end{bmatrix}_{P_0}$$

### 2.3 定理內容

若 $f_1, f_2$ 在臨界點 $P_0:(0,0)$ 的鄰域內連續且有連續偏導數，且 $\det\mathbf{A} \neq 0$，則：

$$\text{非線性系統 } \mathbf{y}' = \mathbf{f}(\mathbf{y}) \text{ 在 } P_0 \text{ 的臨界點類型與穩定性} = \text{線性化系統 } \mathbf{y}' = \mathbf{A}\mathbf{y} \text{ 的結果}$$

### 2.4 例外情況

| 情況 | 可能結果 |
|---|---|
| **A** 有重複特徵值 | 非線性系統可能與線性化相同，也可能為螺旋點 |
| **A** 有純虛數特徵值（線性化為中心） | 非線性系統可能為中心，也可能為螺旋點（**無法確定**） |

> **關鍵限制：** 當線性化結果為「中心」時，不能直接斷言非線性系統也是中心，必須使用其他方法（如 Lyapunov 函數）進一步確認。

## 3. 線性化流程

```
Step 1：找臨界點
        令 f₁ = 0 且 f₂ = 0，解出所有 (y₁, y₂)

Step 2：若臨界點不在原點，平移坐標
        ỹ₁ = y₁ − a，ỹ₂ = y₂ − b

Step 3：計算 Jacobian 矩陣，在臨界點求值得 A
        A = J|_{P₀}

Step 4：對線性化系統 y' = Ay
        計算 p = tr(A)，q = det(A)，Δ = p² − 4q
        套用 Table 4.1 與 Table 4.2 判斷類型與穩定性

Step 5：若線性化結果不是中心（p ≠ 0）
        → 非線性系統與線性化相同（定理保證）
        若線性化結果為中心（p = 0）
        → 需進一步分析（可能是中心或螺旋點）
```

## 4. 範例一：自由無阻尼單擺（Example 1）

### 4.1 物理模型

質量 m、擺長 L 的單擺，忽略空氣阻力與擺桿質量：

$$mL\theta'' + mg\sin\theta = 0 \implies \theta'' + k\sin\theta = 0, \quad k = \frac{g}{L}$$

### 4.2 轉換為一階系統

令 $y_1 = \theta$，$y_2 = \theta'$：

$$\begin{cases} y_1' = y_2 \\ y_2' = -k\sin y_1 \end{cases}$$

### 4.3 求臨界點

令 $y_2 = 0$ 且 $\sin y_1 = 0$：

$$\text{臨界點：} (n\pi, 0), \quad n = 0, \pm 1, \pm 2, \ldots$$

物理意義：
- $n$ 為偶數（$0, \pm 2\pi, \pm 4\pi, \ldots$）→ 擺垂在**最低點**（自然平衡位置）
- $n$ 為奇數（$\pm\pi, \pm 3\pi, \ldots$）→ 擺直立在**最高點**（不穩定平衡）

### 4.4 臨界點 $(0, 0)$ 的線性化

Maclaurin 展開：$\sin y_1 \approx y_1$

Jacobian 矩陣：

$$\mathbf{A} = \begin{bmatrix} 0 & 1 \\ -k & 0 \end{bmatrix}$$

計算：$p = 0,\; q = k > 0,\; \Delta = -4k < 0$

結論：**中心（Center），穩定**

> 由於 $p = 0$（例外情況），定理無法保證非線性系統也是中心。但可用能量積分法證明確實為中心（見 4.5 節 Example 4）。

所有偶數倍 $\pi$ 的臨界點（$n = 0, \pm2, \pm4, \ldots$）均為中心。

### 4.5 臨界點 $(\pi, 0)$ 的線性化

平移：令 $\tilde{y}_1 = y_1 - \pi$，利用：

$$\sin\theta = \sin(\tilde{y}_1 + \pi) = -\sin\tilde{y}_1 \approx -\tilde{y}_1$$

Jacobian 矩陣：

$$\mathbf{A} = \begin{bmatrix} 0 & 1 \\ k & 0 \end{bmatrix}$$

計算：$p = 0,\; q = -k < 0$

結論：**鞍點（Saddle Point），不穩定**

所有奇數倍 $\pi$ 的臨界點（$n = \pm1, \pm3, \ldots$）均為鞍點。

### 4.6 相平面圖解讀

![[4.5.1.png]]

- 中心附近：封閉橢圓形軌跡 → 小角度擺動（振盪）
- 鞍點處：軌跡繞過而不聚集 → 擺翻越最高點的臨界狀態
- $C > k$ 的軌跡：波浪形曲線 → 擺持續旋轉（能量足夠翻越頂點）

## 5. 範例二：阻尼單擺（Example 2）

### 5.1 方程式

加入阻尼項 $c\theta'$：

$$\theta'' + c\theta' + k\sin\theta = 0$$

一階系統：

$$\begin{cases} y_1' = y_2 \\ y_2' = -k\sin y_1 - cy_2 \end{cases}$$

臨界點仍為 $(n\pi, 0)$。

### 5.2 臨界點 $(0, 0)$ 的線性化

$$\mathbf{A} = \begin{bmatrix} 0 & 1 \\ -k & -c \end{bmatrix}$$

$$p = -c < 0, \quad q = k > 0, \quad \Delta = c^2 - 4k$$

| 阻尼大小 | Δ 值 | 臨界點類型 | 穩定性 |
|---|---|---|---|
| 小阻尼 $c^2 < 4k$ | $\Delta < 0$ | 穩定螺旋點 | 漸近穩定 |
| 臨界阻尼 $c^2 = 4k$ | $\Delta = 0$ | 穩定節點 | 漸近穩定 |
| 過阻尼 $c^2 > 4k$ | $\Delta > 0$ | 穩定節點 | 漸近穩定 |

> 只要有阻尼（$c > 0$），$p = -c < 0$，平衡點（最低點）一定為**漸近穩定**。

### 5.3 臨界點 $(\pi, 0)$ 的線性化

$$\mathbf{A} = \begin{bmatrix} 0 & 1 \\ k & -c \end{bmatrix}$$

$$p = -c, \quad q = -k < 0$$

結論：$q < 0$ → **鞍點（Saddle Point），不穩定**（與無阻尼情況相同）。

> 阻尼不改變「頂點」的不穩定性，只影響「底部」的收斂速度。

## 6. 範例三：Lotka-Volterra 捕食者-獵物模型（Example 3）

### 6.1 模型建立

設 $y_1(t)$ = 獵物（兔子）數量，$y_2(t)$ = 捕食者（狐狸）數量：

$$\begin{cases} y_1' = ay_1 - by_1 y_2 & \text{（兔子：自然增長，被吃減少）} \\ y_2' = -ly_2 + ky_1 y_2 & \text{（狐狸：自然死亡，吃兔子增加）} \end{cases}$$

參數均為正數：$a, b, k, l > 0$

### 6.2 求臨界點

$$f_1 = y_1(a - by_2) = 0 \implies y_1 = 0 \text{ 或 } y_2 = a/b$$
$$f_2 = y_2(ky_1 - l) = 0 \implies y_2 = 0 \text{ 或 } y_1 = l/k$$

兩個臨界點：

| 臨界點 | 物理意義 |
|---|---|
| $(0, 0)$ | 兩族群均滅絕 |
| $(l/k,\; a/b)$ | 兩族群共存的平衡狀態（自然之衡） |

### 6.3 臨界點 $(0, 0)$ 的線性化

忽略非線性項 $by_1y_2$ 和 $ky_1y_2$：

$$\mathbf{A} = \begin{bmatrix} a & 0 \\ 0 & -l \end{bmatrix} \implies \lambda_1 = a > 0,\; \lambda_2 = -l < 0$$

$q = -al < 0$ → **鞍點（Saddle Point），不穩定**

> 物理意義：若兩族群數量均極少，系統處於不穩定狀態——獵物會增長，捕食者會衰亡。

### 6.4 臨界點 $(l/k, a/b)$ 的線性化

平移坐標：$\tilde{y}_1 = y_1 - l/k$，$\tilde{y}_2 = y_2 - a/b$

展開並忽略二次非線性項：

$$\tilde{y}_1' = -\frac{lb}{k}\tilde{y}_2, \quad \tilde{y}_2' = \frac{ak}{b}\tilde{y}_1$$

線性化矩陣：

$$\mathbf{A} = \begin{bmatrix} 0 & -lb/k \\ ak/b & 0 \end{bmatrix}$$

$$p = 0, \quad q = \frac{ak \cdot lb}{k \cdot b} = al > 0, \quad \Delta = -4al < 0$$

線性化結果：**中心**

由能量積分法可得封閉軌跡族（橢圓）：

$$\frac{ak}{b}\tilde{y}_1^2 + \frac{lb}{k}\tilde{y}_2^2 = \text{const}$$

可嚴格證明非線性系統也是**中心**（封閉軌跡，但非橢圓）。

**Lotka-Volterra 定性結論：**
- 族群不會達到固定常數，而是永久週期振盪
- 兔子數量高峰 → 狐狸增加 → 兔子減少 → 狐狸減少 → 兔子回升（循環）

## 7. 相平面降階法（Transformation to First-Order Equation）

### 7.1 方法

對二階自治 ODE $F(y, y', y'') = 0$，令 $y_1 = y$，$y_2 = y'$，則：

$$y'' = y_2' = \frac{dy_2}{dt} = \frac{dy_2}{dy_1}\cdot\frac{dy_1}{dt} = \frac{dy_2}{dy_1}\cdot y_2$$

ODE 化為一階方程：

$$F\!\left(y_1,\; y_2,\; \frac{dy_2}{dy_1}y_2\right) = 0$$

### 7.2 應用：自由無阻尼單擺的能量積分（Example 4）

由 $\theta'' + k\sin\theta = 0$，令 $y_2 = \theta'$：

$$\frac{dy_2}{dy_1}y_2 = -k\sin y_1 \implies y_2\, dy_2 = -k\sin y_1\, dy_1$$

積分：

$$\frac{1}{2}y_2^2 = k\cos y_1 + C$$

乘以 $mL^2$（物理詮釋）：

$$\underbrace{\frac{1}{2}m(L\theta')^2}_{\text{動能}} - \underbrace{mL^2 k\cos\theta}_{\text{位能}} = \text{const} \quad \text{（能量守恆）}$$

> **物理意義：** 積分常數 C 代表系統的機械能。不同 C 值對應不同的軌跡族，C 足夠大時擺可越過頂點持續旋轉。

## 8. 本節方法比較

| 方法 | 適用情況 | 優點 | 限制 |
|---|---|---|---|
| **線性化（Jacobian）** | 分析臨界點附近行為 | 簡單快速，直接套 Table 4.1/4.2 | 中心情況不可靠；只描述局部行為 |
| **相平面降階法** | 二階自治 ODE | 可得相平面上的精確軌跡方程 | 需要積分，不一定有解析解 |
| **Lyapunov 函數法** | 全局穩定性分析 | 可確認線性化無法判斷的情況 | 需要找到合適的 Lyapunov 函數（有難度） |

## 9. 臨界點類型對照（含線性化例外）

| 線性化結果 | 可靠性 | 非線性系統實際類型 |
|---|---|---|
| 鞍點 | ✅ 可靠 | 一定是鞍點 |
| 節點（Improper/Proper） | ✅ 可靠 | 一定是節點（同穩定性） |
| 螺旋點 | ✅ 可靠 | 一定是螺旋點（同穩定性） |
| 中心（$p = 0$） | ❌ **不可靠** | 可能是中心，也可能是螺旋點 |
| 退化節點 | ⚠️ 部分可靠 | 可能是節點或螺旋點 |

## 重點整理

| 概念 | 關鍵內容 |
|---|---|
| 自治系統 | 右側不含 t；臨界點為 $\mathbf{f}(\mathbf{y}) = \mathbf{0}$ |
| 線性化矩陣 | Jacobian 在臨界點求值：$\mathbf{A} = \mathbf{J}\|_{P_0}$ |
| 線性化定理 | $\det\mathbf{A} \neq 0$ 且結果非中心 → 非線性系統與線性化一致 |
| 中心的不確定性 | $p = 0$ 時無法用線性化確定，需能量法或 Lyapunov 函數 |
| 單擺（無阻尼） | 底部 $(2n\pi, 0)$ → 中心；頂部 $((2n+1)\pi, 0)$ → 鞍點 |
| 單擺（有阻尼） | 底部 → 穩定螺旋/節點（漸近穩定）；頂部 → 鞍點（不穩定） |
| Lotka-Volterra | $(0,0)$ → 鞍點；$(l/k, a/b)$ → 中心（族群週期振盪） |
| 相平面降階 | $y'' = (dy_2/dy_1) \cdot y_2$，化二階為一階，積分得軌跡方程 |
