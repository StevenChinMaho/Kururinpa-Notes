## 1. 基本定義

- **二階線性 ODE (Linear ODEs)**：方程式若能寫成 $y'' + p(x)y' + q(x)y = r(x)$ 的形式，即為線性常微分方程式 。
    
- **非線性 ODE (Nonlinear ODEs)**：若方程式無法寫成上述線性形式，則稱為非線性 。
    
- **齊次 (Homogeneous)**：在標準形式中，若等式右邊的 $r(x) = 0$ (即 $y'' + p(x)y' + q(x)y = 0$)，該方程式即為齊次方程式 。
    
- **非齊次 (Nonhomogeneous)**：若等式右邊的 $r(x) \neq 0$，則稱為非齊次方程式 。
    

## 2. 重疊原理 (Superposition Principle)

- **齊次線性 ODE 基礎定理**：對於齊次線性 ODE，在開區間 $I$ 上的兩個解的任意線性組合 ($y = c_1y_1 + c_2y_2$)，仍然會是該區間上的解 。
    
- 換言之，方程式個別解的總和以及常數倍數，依舊滿足原方程式 。
    
- **重要觀念**：此定理不適用於非齊次線性方程式與非線性方程式，這兩類方程式個別解的總和通常不會是原方程式的解 。
    

## 3. 解的結構與初始值問題 (Initial Value Problem, IVP)

- **初始值問題**：包含一個二階齊次線性 ODE ($y'' + p(x)y' + q(x)y = 0$) 以及兩個初始條件 ($y(x_0) = K_0$ 和 $y'(x_0) = K_1$) 。
    
- **基底 (Basis)**：在開區間 $I$ 上，方程式的任意兩個「線性獨立 (Linearly Independent)」解可以形成一組基底 。線性獨立意味著這兩個解彼此不成比例 (例如：兩函數的商不為常數) 。
    
- **通解 (General Solution)**：由基底所構成的解，形式為 $y = c_1y_1 + c_2y_2$，其中 $c_1$ 與 $c_2$ 為任意常數 。
    
- **特解 (Particular Solution)**：利用初始條件來決定通解中 $c_1$ 與 $c_2$ 的特定數值，藉此得到的唯一解即為特解 。
    

## 4. 降階法 (Reduction of Order)

- **核心思想**：降階法是一個重要的技巧，能將複雜的二階 ODE 簡化為一階 ODE 。
    
- **已知一解求另一解**：如果我們已經知道齊次線性 ODE 的其中一個解 $y_1$，可以令第二個解為 $y = uy_1$ (其中 $u$ 為未知函數) 。
    
- **運算步驟**：將 $y, y', y''$ (如：$y' = u'y_1 + uy_1'$) 代入原方程式並化簡後，再令 $v = u'$，即可將原本的方程式轉換成以 $v$ 為變數的一階 ODE 來求解 。
    
- **適用範圍**：對於一般的二階 ODE $F(x, y, y', y'') = 0$ (無論線性與否)，只要方程式中沒有明確出現 $x$ 或沒有明確出現 $y$，皆可透過令 $z = y'$ 來降階為一階方程式處理 。
