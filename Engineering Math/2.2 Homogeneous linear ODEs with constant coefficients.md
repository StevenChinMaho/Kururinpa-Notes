# 2.2 Homogeneous linear ODEs with constant coefficients

## 1. 核心概念與特徵方程式

- **標準形式**：探討的方程式形式為 $y'' + ay' + by = 0$，其中 $a$ 與 $b$ 為常數 。
    
- **假設解的形式**：因為指數函數的導數仍是指數函數，所以我們可以嘗試將解假設為 $y = e^{\lambda x}$ 。
    
- **特徵方程式 (Characteristic Equation)**：將 $y = e^{\lambda x}$、$y' = \lambda e^{\lambda x}$ 與 $y'' = \lambda^2 e^{\lambda x}$ 代入原方程式後，會得到 $\lambda^2 + a\lambda + b = 0$ 。
    
- **求根公式**：利用一元二次方程式的公式解，可得出特徵方程式的根為 $\lambda_{1,2} = \frac{1}{2}(-a \pm \sqrt{a^2 - 4b})$ 。
    

## 2. 解的三種情況

根據判別式 $a^2 - 4b$ 的正負值，特徵方程式的根會分為三種情況，進而決定通解的形式：

- **情況一：相異實根 (Two real roots)**
    
    - **條件**：$a^2 - 4b > 0$ 。
        
    - **基底**：$y_1 = e^{\lambda_1 x}$ 與 $y_2 = e^{\lambda_2 x}$ 。
        
    - **通解**：$$y = C_1e^{\lambda_1 x} + C_2e^{\lambda_2 x}$$ 
        
- **情況二：實數重根 (A real double root)**
    
    - **條件**：$a^2 - 4b = 0$ 。
        
    - **特徵根**：$\lambda_1 = \lambda_2 = -a/2$
        
    - **尋找第二個解**：為了得到線性獨立的第二個解，需利用「降階法 (Reduction of order)」乘以 $x$ 。
        
    - **基底**：$y_1 = e^{-ax/2}$ 與 $y_2 = xe^{-ax/2}$ 。
        
    - **通解**： $$y = C_1e^{-ax/2} + C_2xe^{-ax/2}$$ 
        
- **情況三：共軛複數根 (Complex conjugate roots)**
    
    - **條件**：$a^2 - 4b < 0$ 。
        
    - **特徵根**：設兩根為 $\lambda = \alpha \pm i\beta$ 。
        
    - **利用尤拉公式轉換**：將複數指數轉換為三角函數，確保解為實數函數 。
        
    - **基底**： $y_{1} = e^{\alpha x}\cos{\beta x}$ 與 $y_{2} = e^{\alpha}\cos{\beta x}$ 。
        
    - **通解**： $$y = e^{\alpha x}(C_{1}\cos{(\beta x)} + C_{2}\sin{(\beta x)})$$ 
        

## 3. 系統不穩定性 (Instability)

- 當系統具有不穩定性時，在初始條件 ($t=0$) 給予極微小差異 (例如 0.001 的改變)，隨著時間 $t$ 的增加，誤差會被指數項 (如 $e^t$) 放大，導致後期結果出現巨大偏差 。
    
- 在實際的工程或電路應用中，這種不穩定性通常是不受歡迎的 。
