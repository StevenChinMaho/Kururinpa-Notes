# 2.4 Components of a Typical x86 Computer

## 1. 主機板 (Motherboard)

主機板是電腦的核心平台，整合了多種關鍵組件 ：

- **核心插槽與槽位**：包含 CPU 插槽、外部快取記憶體槽位，以及主記憶體 (RAM) 插槽 。
    
- **晶片與控制器**：包含 BIOS 晶片、音效合成晶片、影片控制器晶片，以及記憶體控制中心 (Memory controller hub) 。
    
- **擴充與連接**：
    
    - 提供 PCI 匯流排連接器供擴充卡使用 。
        
    - 包含 IDE 磁碟機連接器、電源連接器與軟碟機連接器 。
        
    - 提供 USB、平行埠、序列埠、鍵盤與滑鼠等連接埠 。
        

## 2. 影片輸出 (Video Output)

- **影片控制器**：可內建於主機板或安裝於擴充卡（如 AGP 技術）。
    
- **影片記憶體 (VRAM)**：專門用於處理影像資料，具備雙連接埠 (dual ported) 特性以優化視訊刷新 。
    
- **顯示器類型**：
    
    - **CRT 顯示器**：使用光柵掃描 (raster scanning)，包含水平與垂直回掃 (retrace) 。
        
    - **LCD 顯示器**：直接數位顯示，不需要光柵掃描 。
        

## 3. 記憶體類型 (Memory Types)

系統中使用了多種不同特性的記憶體 ：

- **ROM / EPROM**：唯讀記憶體與可抹除、可程式化的唯讀記憶體 。
    
- **Dynamic RAM (DRAM)**：成本低廉但必須不斷刷新，用於一般動態隨機存取記憶體 。
    
- **Static RAM (SRAM)**：成本昂貴、速度快且不需刷新，主要用於快取記憶體 (Cache) 。
    
- **CMOS RAM**：互補式金屬氧化物半導體，用於儲存 BIOS 系統設定資訊 。
    

## 4. 輸出入連接埠與裝置介面 (I/O Ports & Interfaces)

- **USB (通用序列匯流排)**：支援熱插拔的高速連接方式，速度可達 12 Mbps 。
    
- **平行埠 (Parallel)**：常見於印表機，支援雙向資料傳輸 。
    
- **序列埠 (Serial)**：使用 RS-232 標準，一次傳輸一個位元，適合長距離連線 。
    
- **現代介面**：
    
    - **SATA**：用於硬碟與 CD-ROM 的快速序列介面 。
        
    - **FireWire (IEEE 1394)**：支援多裝置同時連接的高速介面 (800 MB/sec) 。
        
    - **無線技術**：包含低功耗、短距離的 **Bluetooth** 以及速度更快的 **Wi-Fi** (802.11 標準) 。
