## 📌 本節概覽

`INC / DEC` `ADD / SUB` `NEG` `算術運算式` `旗標 Flags`

---

## 1 — INC 和 DEC 指令

- **INC**：destination ← destination + 1
    
- **DEC**：destination ← destination – 1
    
- _備註：運算元可以是暫存器或記憶體位址（自動 dereference）_
    

> **影響旗標**：Overflow、Sign、Zero、Auxiliary Carry、Parity
> 
> **注意：不影響 Carry Flag**

```asm
mov ax, 00FFh
inc ax         ; AX = 0100h
mov ax, 00FFh
inc al         ; AL = 00h  (注意：只對 AL 加，AX 高位被清零)
```

---

## 2 — ADD 和 SUB 指令

語法規則與 MOV 相同（大小需一致、不可 memory→memory）

- **ADD dst, src** → dst ← dst + src
    
- **SUB dst, src** → dst ← dst – src
    

```asm
.data
var1 DWORD 10000h
var2 DWORD 20000h
.code
  mov eax, var1  ; EAX = 00010000h
  add eax, var2  ; EAX = 00030000h
  add ax,  0FFFFh; EAX = 0003FFFFh
  add eax, 1     ; EAX = 00040000h
  sub ax,  1     ; EAX = 0004FFFFh
```

---

## 3 — NEG 指令（取負數）

反轉運算元的正負號，運算元可為暫存器或記憶體。

```asm
neg al        ; AL: -1 → +1
neg valW      ; valW: +32767 → -32767
```

- **任何非零運算元執行 NEG → CF = 1**
    
- **若無法產生正確結果（如 NEG -128）→ OF = 1**

## 4 — 實作算術運算式

_範例：Rval = –Xval + (Yval – Zval)，其中 Xval=26, Yval=30, Zval=40_

```asm
mov eax, Xval      ; EAX = 26
neg eax            ; EAX = -26
mov ebx, Yval      ; EBX = 30
sub ebx, Zval      ; EBX = -10
add eax, ebx       ; EAX = -36
mov Rval, eax      ; Rval = -36
```

> **技巧**：若不希望修改原始變數，先將值搬入暫存器再運算。

## 5 — 算術旗標（Flags Affected by Arithmetic）

ALU 依目的運算元的結果設定旗標；MOV 指令永遠不影響旗標。

### ZF — Zero Flag

結果等於 0 時設為 1

```asm
sub ecx, 1  ; ECX=0 → ZF=1
inc eax     ; EAX=1 → ZF=0
```

### SF — Sign Flag

結果為負數時設為 1（即目的運算元最高位元的複製）

```asm
sub eax, 5  ; EAX=-1 → SF=1
```

### CF — Carry Flag

- **加法**：最高位產生進位時設為 1（視為第 9 個位元）
    
- **減法**：借位時設為 1（= NOT 借位訊號）
    

```asm
mov al, 0FFh
add al, 1   ; CF=1, AL=00h
mov al, 1
sub al, 2   ; CF=1, AL=FFh
```

### OF — Overflow Flag

有符號整數溢位時設為 1

正 + 正 → 負，或 負 + 負 → 正

```asm
mov al, +127
add al, 1   ; OF=1 (正→負)
mov al, -128
sub al, 1   ; OF=1 (負→正)
```

### AF — Auxiliary Carry

低 4 位元（low nibble）向高 4 位元進位或借位時設為 1

```asm
mov al, 0Fh
add al, 1   ; AC=1
```

### PF — Parity Flag

目的運算元最低位元組中 1 的數量為偶數時設為 1

```asm
add al, 2   ; AL=10001110b → PF=1
sub al, 80h ; AL=00001110b → PF=0
```

## 6 — 旗標快速判斷口訣

- **CF（無符號溢位）**：加法看最高位元進位；減法看是否借位（結果取 NOT）
    
- **OF（有符號溢位）**：正＋正＝負，或 負＋負＝正 → OF=1；兩數符號不同相加，OF 永遠為 0
    
- **硬體偵測 OF**：CF XOR 次高位元的進位；兩者不同時 OF=1
    
- **NEG 特殊情形**：-128 取 NEG → CF=1, OF=1（無法表示 +128）
