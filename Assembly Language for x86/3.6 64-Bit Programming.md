## 一、32-bit vs 64-bit 程式結構差異

### 32-bit 版本（原始）

```asm
.386
.model flat, stdcall
.stack 4096
ExitProcess PROTO, dwExitCode:DWORD

.data
sum DWORD 0

.code
main PROC
    mov eax, 5
    add eax, 6
    mov sum, eax
    INVOKE ExitProcess, 0
main ENDP
END main
```

### 64-bit 版本

```asm
ExitProcess PROTO          ; 不需要參數型態宣告

.data
sum QWORD 0                ; DWORD → QWORD

.code
main PROC
    mov rax, 5             ; eax → rax
    add rax, 6
    mov sum, rax
    mov ecx, 0
    call ExitProcess       ; INVOKE → mov + call
main ENDP
END                        ; 不需要指定程式進入點
```

---

## 二、主要差異整理

|項目|32-bit|64-bit|
|---|---|---|
|CPU 模式宣告|`.386`|不需要|
|記憶體模型|`.model flat, stdcall`|不需要|
|堆疊宣告|`.stack 4096`|不需要|
|ExitProcess 宣告|`PROTO, dwExitCode:DWORD`|`PROTO`（無參數）|
|呼叫結束程序|`INVOKE ExitProcess, 0`|`mov ecx, 0` + `call ExitProcess`|
|程式進入點|`END main`|`END`（不指定）|
|資料型態|`DWORD`|`QWORD`|
|暫存器|`eax`|`rax`|

---

## 三、暫存器對照

64-bit 模式將原本 32-bit 暫存器擴充為 64-bit，命名前綴由 `E` 改為 `R`：

|32-bit|64-bit|
|---|---|
|EAX|**RAX**|
|EBX|**RBX**|
|ECX|**RCX**|
|EDX|**RDX**|
|ESI|**RSI**|
|EDI|**RDI**|
|ESP|**RSP**|
|EBP|**RBP**|

---

## 四、INVOKE 指令的差異

64-bit 版本的 MASM **不支援 INVOKE 指令**，需改用暫存器傳遞參數：

```asm
; 32-bit 寫法
INVOKE ExitProcess, 0

; 64-bit 對應寫法
mov ecx, 0        ; 第一個參數放入 ecx
call ExitProcess
```

---

## 五、執行環境需求

- 需要在 **64-bit Windows** 上執行
- 需要 **Visual Studio 2012 或更新版本**

---

## 六、完整範例對照

```asm
;========== 32-bit ==========        ;========== 64-bit ==========
.386                                  ;（移除）
.model flat,stdcall                   ;（移除）
.stack 4096                           ;（移除）
ExitProcess PROTO,                    ExitProcess PROTO
  dwExitCode:DWORD
.data                                 .data
sum DWORD 0                           sum QWORD 0
.code                                 .code
main PROC                             main PROC
  mov eax,5                             mov rax,5
  add eax,6                             add rax,6
  mov sum,eax                           mov sum,rax
  INVOKE ExitProcess,0                  mov ecx,0
                                        call ExitProcess
main ENDP                             main ENDP
END main                              END
```

---

## 七、重點整理

```
64-bit 不需要 .386 / .model / .stack 宣告
暫存器由 E 開頭改為 R 開頭（eax → rax）
資料型態建議改用 QWORD（64-bit）
不支援 INVOKE，改用 mov + call 傳遞參數
END 後面不需指定程式進入點
需要 64-bit Windows + Visual Studio 2012+
```
