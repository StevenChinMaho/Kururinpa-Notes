# 4.5 JMP and LOOP Instructions

## 本節涵蓋主題

|主題|說明|
|---|---|
|`JMP`|無條件跳躍指令|
|`LOOP`|計數型迴圈指令（自動使用 ECX）|
|LOOP 範例|計算 5+4+3+2+1|
|在迴圈內修改 ECX|潛在無限迴圈與安全處理方式|
|巢狀迴圈|儲存 / 還原外層 ECX|
|綜合範例|陣列加總、字串複製|

---

## 1. JMP 指令

**無條件跳躍**到指定的程式碼標籤，通常在同一個 procedure 內使用。

### 語法

```asm
JMP destination
```

- **邏輯**：`EIP ← destination`（將指令指標設為目標標籤的位址）
- 跳躍到 **目前 procedure 之外**，必須使用特殊的 **global label**（詳見 Section 5.5.2.3）

### 範例：無限迴圈

```asm
top:
    .           ; 迴圈主體
    .
    jmp top     ; 無條件跳回 top
```

---

## 2. LOOP 指令

建立**計數型迴圈**，自動使用 **ECX** 作為計數器。

### 語法

```asm
LOOP destination
```

### 執行邏輯

1. `ECX ← ECX – 1`
2. 若 `ECX ≠ 0`，跳躍到 `destination`；若 `ECX = 0`，繼續執行下一行

### 限制

- 跳躍目標（destination）必須在目前位置計數器的 **−128 到 +127 bytes** 之內
- 若迴圈主體過大，MASM 會產生錯誤：

```
error A2075: jump destination too far: by 14 byte(s)
```

---

## 3. LOOP 範例：計算 5 + 4 + 3 + 2 + 1

```asm
mov ax, 0      ; AX = 0（累加器）
mov ecx, 5     ; ECX = 5（迴圈次數）
L1:
    add ax, cx ; AX += CX（5, 4, 3, 2, 1 依序累加）
    loop L1    ; ECX -= 1；ECX ≠ 0 則跳回 L1
; 結束：AX = 15
```

### 機器碼與位址解析

```
offset    machine code     source code
00000000  66 B8 0000       mov ax, 0
00000004  B9 00000005      mov ecx, 5
00000009  66 03 C1     L1: add ax, cx
0000000C  E2 FB            loop L1
0000000E                   ← LOOP 組譯後的「下一指令位址」
```

LOOP 組譯時，目前位置計數器 = `0000000Eh`，加上偏移量 `FBh`（= −5）：

```
0000000Eh + FBh = 00000009h  ← 跳回 L1
```

## 4. Your Turn 練習

### 練習一：AX 最終值為何？

```asm
mov ax, 6
mov ecx, 4
L1:
    inc ax
    loop L1
; ECX 從 4 遞減到 0，inc ax 執行 4 次
; AX = 6 + 4 = 10
```

### 練習二：迴圈執行幾次？

```asm
mov ecx, 0    ; ECX = 0
X2:
    inc ax
    loop X2
```

> ECX 初始為 0，LOOP 先將 ECX − 1 = `FFFFFFFFh`（≠ 0），因此迴圈執行 **4,294,967,296 次**（= `FFFFFFFFh + 1`）。
> 
> **結論**：LOOP 前必須確保 ECX > 0，否則會誤觸超大次數迴圈。

## 5. 在迴圈內修改 ECX

LOOP 允許在迴圈主體內修改 ECX，但若不小心可能造成 ECX 永遠不等於 0，產生**無限迴圈**。

### 錯誤示範（無限迴圈）

```asm
top:
    .
    inc ecx       ; ECX 持續增加，永遠不會歸零
    loop top      ; 無限迴圈！
```

### 安全做法：用變數儲存 / 還原 ECX

在迴圈開頭將 ECX 存入記憶體變數，在 LOOP 指令前還原。

```asm
.data
count DWORD ?

.code
    mov ecx, 100    ; 外層迴圈次數
top:
    mov count, ecx  ; 儲存目前的 ECX
    .
    mov ecx, 20     ; 在迴圈內使用 ECX 做其他計算
    .
    mov ecx, count  ; 還原 ECX
    loop top
```

## 6. 巢狀迴圈（Nested Loop）

外層迴圈的 ECX 值在進入內層迴圈前必須先**儲存**，內層結束後**還原**，才能讓外層迴圈正確計數。

```asm
.data
count DWORD ?       ; 用來暫存外層 ECX

.code
    mov ecx, 100    ; 外層迴圈執行 100 次
L1:
    mov count, ecx  ; 儲存外層計數器
    mov ecx, 20     ; 內層迴圈執行 20 次
L2:
    .               ; 內層迴圈主體
    loop L2         ; 內層：ECX -= 1，若 ≠ 0 跳回 L2
    mov ecx, count  ; 還原外層計數器
    loop L1         ; 外層：ECX -= 1，若 ≠ 0 跳回 L1
; 總執行次數：100 × 20 = 2,000 次
```

## 7. 綜合範例

### 範例一：陣列加總（Summing an Integer Array）

計算 DWORD 陣列所有元素的總和，搭配 `OFFSET`、`LENGTHOF`、`TYPE`。

```asm
.data
intarray DWORD 10000h, 20000h, 30000h, 40000h

.code
    mov edi, OFFSET intarray    ; EDI 指向陣列起始位址
    mov ecx, LENGTHOF intarray  ; ECX = 4（迴圈次數）
    mov eax, 0                  ; 清空累加器
L1:
    add eax, [edi]              ; EAX += 目前元素
    add edi, TYPE intarray      ; EDI += 4，指向下一個元素
    loop L1                     ; ECX -= 1；≠ 0 則跳回 L1
; 結束：EAX = A0000h
```

### 範例二：複製字串（Copying a String）

使用索引定址逐字元複製，`SIZEOF` 同時決定目標緩衝區大小與迴圈次數。

```asm
.data
source BYTE "This is the source string", 0
target BYTE SIZEOF source DUP(0)    ; 配置與 source 等大的空間（含 null）

.code
    mov esi, 0               ; 索引從 0 開始
    mov ecx, SIZEOF source   ; ECX = 字串總長（含 null 結尾）
L1:
    mov al, source[esi]      ; 從 source 取一個字元
    mov target[esi], al      ; 寫入 target
    inc esi                  ; 移到下一個字元
    loop L1                  ; ECX -= 1；≠ 0 則繼續
```

## 重點整理

|指令|功能|注意事項|
|---|---|---|
|`JMP label`|無條件跳躍|跨 procedure 需用 global label|
|`LOOP label`|ECX -= 1；≠ 0 則跳|跳躍範圍限 ±128 bytes；ECX = 0 時會執行 2³² 次|
|儲存 / 還原 ECX|迴圈內修改 ECX 的安全做法|用記憶體變數在迴圈開頭儲存、LOOP 前還原|
|巢狀迴圈|多層迴圈|每層都需獨立儲存 / 還原 ECX|
|`add edi, TYPE array`|指標步進|步距必須與元素型別大小一致|

> **ECX 陷阱**：`LOOP` 指令**先遞減後判斷**，因此 ECX 初始值為 0 時，會誤以為需執行 4,294,967,296 次，務必在進入迴圈前確保 ECX > 0。
